# Глоссарий локализации (русский) — консистентность с CredosCRM

**Дата:** 2026-06-20
**Источник:** русификация платформы из CredosCRM (`../../research/credos-crm-i18n/` — Lingui `ru-RU.po`: front 17k строк, server 8k, emails).
**Назначение:** наши UI-ярлыки (`tt*`) должны совпадать с терминологией платформы Twenty/CRM, чтобы при установке в общий workspace всё звучало единообразно.

## ⚠️ Расхождения терминов (НЕ путать с платформой)

| Термин | Платформа Twenty | У НАС (time) | Почему |
|---|---|---|---|
| **Activity** | «Активность» (лента событий записи) | **«Вид работ»** (`ttActivity`) | у нас это справочник видов работ, НЕ activity-лента |
| **Project** | (в CRM может быть `credosProject` — сделка/поставка) | **«Проект»** (`ttProject`, учёт времени) | разные сущности; нейминг `tt` отделяет |
| **Member** | «Участник» (workspaceMember) | **«Работник»/«Сотрудник»** (`ttEmployee`) | наш кадровый объект |

Эти термины переопределяем своими русскими ярлыками (в `labels.ts`), не наследуем платформенные.

## Правило
1. Общие UI-слова — терминология платформы (см. таблицу выше / `ru-RU.po`).
2. Доменные сущности time — наши ярлыки из `labels.ts`, даже если платформа переводит иначе (Activity).
3. При новом UI-тексте — сверяйся с `../../research/credos-crm-i18n/front-ru-RU.po` (поиск по msgid).
