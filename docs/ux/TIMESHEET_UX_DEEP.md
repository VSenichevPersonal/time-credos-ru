# Timesheet UX Deep Dive (референс Timetta)

**Дата:** 2026-06-21

---

## Primary Input Mode
- **Основной: ежедневный список** (не недельная сетка!)
- Таймер: ✅ есть (старт/стоп, чёрный треугольник в строке)
- Ретро-ввод: ✅ поддерживается (заполнение после факта)
- Real-time: ✅ через таймер
- Для консалтинга: гибкое переключение режимов

## Гранулярность строки
- **Проект × задача** (ProjectTask), не просто WorkType
- WorkType = дополнительная аналитика
- Настройка через шаблон таймшита (какие колонки показывать)
- Предотвращение взрыва строк: через шаблоны + авто-подстановку

## Снижение времени ввода (что реально работает)
1. **Копирование прошлой недели** — самый популярный метод
2. **Шаблоны таймшитов** — предзаполненные строки проектов/задач
3. **Авто-подстановка из истории** — предлагает недавние комбинации
4. **Кнопки-пресеты** (8, 4, 0.5) — один клик
5. **Таймер** — для задач в реальном времени

## Reopen согласованного
- ❌ НЕЛЬЗЯ редактировать согласованный таймшит
- Только через спец-процедуру (администратор)
- После отклонения: авто DRAFT, можно править
- Нет версионирования таймшитов

## Норма в сетке
- ❌ План/норма НЕ показывается прямо в сетке таймшита
- Контроль через: аналитику, отчёты, визуальные подсказки
- Отклонение видно в отдельных отчётах
- Мотивация: уведомления + просроченные таймшиты

## Inline-валидации
- Блокировка: зависит от настроек правил валидации
- Предупреждение: день ±2ч от расписания
- Дубли строк: НЕ блокируются (разрешены)
- Отпуск/выходной: проверка через календарь
- 16ч/день: правило «Рабочий день до 24 часов»

## План vs Факт
- В таймшите: НЕТ прямого сравнения
- В отчётах: ✅ (Ресурсный план, Plan/Estimate/Actual)
- В дашборде: ✅ (Сводка по сотрудникам)
- Подсветка отклонения: цветами в capacity board

## Billable флаг
- Уровни: проект / вид работ / строка таймшита
- В UI: сотрудник видит разницу (оплачиваемые/неоплачиваемые)
- Наследование: проект → задача → строка

## Перф сетки + Autosave
- Автосохранение: on blur (мгновенно при потере фокуса) «Сохранить»
- НЕ ручное — мгновенное на blur. Нет кнопки Сохранить.
- Виртуализация: для больших сеток
