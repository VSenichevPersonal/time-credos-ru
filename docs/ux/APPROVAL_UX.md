# Approval UX Flow (референс)

**Источник:** документация
**Дата:** 2026-06-21

---

## Batch Approve (массовое согласование)

- ✅ Поддерживается
- Выбор нескольких таймшитов (чекбоксы)
- Кнопка «Согласовать всё»
- Ускоряет работу с большой командой

## Отклонение

- ✅ Комментарий ОБЯЗАТЕЛЕН
- Без комментария кнопка «Отклонить» неактивна
- Сотрудник видит причину полностью
- Места отображения: карточка таймшита, список «Отправленные», email, уведомления
- Статус после отклонения: возврат в «Черновик»

## Уведомления при смене статуса

- ✅ Email + push-уведомления
- События: Отправлен, Согласован, Отклонён
- Кому: автору таймшита + руководителю
- Мгновенно при смене статуса

## Создание Акта

- Простая форма (НЕ wizard)
- Два способа создания: через Финансы/Проекты или через карточку проекта → вкладка Выручка
- Поля: проект, номер (авто), описание
- Строки акта добавляются вручную
- ActLine: работа, количество, цена, учётная статья (выбор вручную)
- Автозаполнения из таймшита НЕТ
