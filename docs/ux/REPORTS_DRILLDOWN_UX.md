# Reports & Drill-down UX (референс)

**Источник:** документация
**Дата:** 2026-06-21

---

## Drill-down механика

- Клик на число → **новая страница** (не модалка, не вкладка)
- Детализация в виде таблицы со строками
- Есть хлебные крошки (навигация назад)
- Можно фильтровать внутри детализации
- Максимальная глубина: до конкретного таймшита

## Группировка

- ✅ Переключение группировки в одном отчёте
- Оси: отдел, проект, сотрудник, период
- Cross-filter: настраивается через построитель отчётов

## Export

- ✅ Excel и CSV из любого уровня drill-down
- Иерархия сохраняется при экспорте

## Дополнения

### Фильтры (H3)
- Период (дата с/по)
- Сотрудник
- Отдел
- Проект
- Клиент (организация)
- Статус таймшита
- WorkType (вид работ)
- Роль
- Программа
- Ресурсный пул

### Период (H4)
- ✅ Неделя / Месяц / Квартал / Год
- Переключение через настройку отчёта (не прямой dropdown)
- Доступно в большинстве аналитических отчётов

### Scheduled Reports (G4)
- ❌ Нет авто-рассылки по расписанию
- Экспорт только вручную
