# Report Columns — Detail уровень (референс)

**Источник:** «Таймшиты детально» — конечный уровень drill-down
**Дата:** 2026-06-21

---

## Группы колонок

### Дата и период
| Поле | Код |
|------|-----|
| Дата | Date |

### Сотрудник и подразделение
| Поле | Код |
|------|-----|
| Пользователь | UserId |
| Подразделение | DepartmentId |
| Руководитель | SupervisorId |
| Грейд | GradeId |
| Уровень | LevelId |
| Локация | LocationId |
| Юрлицо пользователя | UserLegalEntity |

### Организация и ресурсы
| Поле | Код |
|------|-----|
| Организация (клиент) | OrganizationId |
| Менеджер организации | OrganizationManagerId |
| Ресурсный пул | ResourcePoolId |
| Менеджер ресурсного пула | ResourcePoolManagerName |
| Субподрядчик | SubcontractorId |

### Проект и задача
| Поле | Код |
|------|-----|
| Проект | ProjectId |
| Юрлицо проекта | ProjectLegalEntity |
| Менеджер проекта | ProjectManagerId |
| Работа (задача) | ProjectTaskId |
| Этап (работа верхнего уровня) | FirstLevelTaskId |
| Программа | ProgramId |
| Менеджер программы | ProgramManagerId |

### Роль и учёт
| Поле | Код |
|------|-----|
| Роль | RoleId |
| Вид работ | WorkType |
| Учётная статья | FinancialAccount |
| Код оплаты | BillCode |
| Тип биллинга | BillingType |

### Часы
| Поле | Код |
|------|-----|
| Часы | Duration (Decimal) |
| Оплачиваемые часы | BillableDuration |
| Неоплачиваемые часы | NonBillableDuration |

### Статус и документы
| Поле | Код |
|------|-----|
| Статус таймшита | TimeSheetState |
| Комментарий | Description |
| Статус экспорта | Exported |

---

## Что берём для своего detail-уровня

1. Минимум: Дата + Сотрудник + Проект + Задача + Часы + Статус
2. Стандарт: + Отдел + Клиент + Роль + WorkType
3. Максимум: все 40+ колонок через конфигурацию отчёта
