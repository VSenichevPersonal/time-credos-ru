# RBAC Model (референс Timetta)

**Дата:** 2026-06-21

---

## Встроенные роли

| Роль | Область | Ключевые права |
|------|---------|---------------|
| Администратор | Система | Полный доступ, настройки, пользователи |
| Управление проектами | Проекты | Создание/редактирование проектов, Gantt |
| Управление командой | Команда | Просмотр/согласование таймшитов подчинённых |
| Управление ресурсами | Ресурсы | Бронирование, планирование загрузки |
| Управление финансами | Финансы | P&L, ставки, рентабельность, счета |
| Управление клиентами | Клиенты | CRM, сделки, воронка |
| Пользователь | Базовый | Свой таймшит, свои задачи |

Иерархия: НЕТ жёсткой иерархии. Ролевая модель на основе областей доступа + гранул.

## Видимость таймшитов

- **Сотрудник:** только свой таймшит
- **Руководитель:** таймшиты своих подчинённых (через «Моя команда»)
- **Менеджер нескольких отделов:** видит все свои отделы
- **Администратор:** все таймшиты

## Согласование

- **По умолчанию:** прямой руководитель
- **Менеджер проекта:** НЕ может одобрять (только если включён в воркфлоу)
- **Делегирование:** через настройку воркфлоу (Lifecycle → роли перехода)

## Финансовый доступ

- **P&L, ставки, рентабельность:** роль «Управление финансами»
- **Своя ставка:** НЕ видна сотруднику (только часы)
- **Cost-rate vs bill-rate:** раздельные права просмотра
- **Field-level security:** ставки закрыты от рядовых пользователей
