# REQ-0002 «Финансы» — Acceptance Criteria

**Источник:** P&L-алгоритм референсной системы (документация, проверено)
**Дата:** 2026-06-21

---

## AC-1: Расчёт P&L

### Формулы
- `Баланс = Выручка − Затраты (включаемые в баланс)`
- `Непогашенные затраты = Все затраты − Выручка`
- `Финальный P&L = Баланс − CCR (затраты на капитал)`

### Источники данных
| Данные | Источник |
|--------|----------|
| Выручка | Оценки выручки + фактические акты |
| Оплаты | Поступления от клиента |
| Затраты | Проводки (таймшиты, субподряд, материалы) |
| Себестоимость труда | Авто из согласованных таймшитов |
| CCR | Настройка проекта (ставка % годовых) |

### 4 режима отображения
- **План** — min/max дат оценок из мастер-версии проекта
- **Оценка** — min/max дат оценок из текущей версии проекта
- **Факт** — min/max дат проводок и оплат
- **Прогноз** — объединение периодов Факта и Оценки

### AC-1.1: Период P&L
- Период определяется по датам проводок и оплат, НЕ по датам проекта
- Пример: проект 01.01–31.12, первая проводка 15.02, последняя оценка 10.11 → период P&L = 15.02–10.11

### AC-1.2: Права доступа
- Роль «Финансовый менеджер»
- Гранула «Финансовые сведения»

---

## AC-2: P&L по уровням

P&L строится для:
- Отдельного проекта
- Программы
- Клиента
- Портфеля

---

## AC-3: Ставки и себестоимость

### Два типа себестоимости
- Управленческая — для планирования (внутренняя ставка)
- Бухгалтерская — фактическая по данным бухгалтерии

### Связь с таймшитами
- При согласовании таймшита → авто-проводки:
  - Статья «Себестоимость труда» (по проектам и задачам)
  - Статья «Себестоимость отсутствий» (для оплачиваемых отсутствий)
- Баланс отсутствий: записи с типом «Начисление»

---

## AC-4: Валидация

### AC-4.1: Формулы
- Баланс ≥ 0 при Выручке > Затрат
- Непогашенные затраты < 0 = профицит (долг закрыт)
- Непогашенные затраты > 0 = дефицит

### AC-4.2: Данные
- Период P&L ⊆ период существования проводок
- Только согласованные таймшиты учитываются в себестоимости
