## 📊 Данные для MVP

### Источник данных
- Каталог услуг (DOCX → ~42 услуги)
- Список продуктов из каталога (~25 продуктов)
- Структура отделов (4 отдела, категории)

### Структура отделов

```
📁 Отдел информационной безопасности (ОИБ)
├── Аудит (6 услуг)
├── Документационное обеспечение (4 услуги)
├── Внедрение СЗИ (1 услуга, 25+ продуктов)
├── Оценка соответствия и аттестация (4 услуги)
├── Техническая поддержка (2 услуги)
└── КИИ (5 услуг)

📁 Отдел практической ИБ (ОПИБ)
└── 8 услуг (Pentest, мониторинг, MSSP...)

📁 Технический центр
└── 8 услуг (ИТ-аутсорсинг, миграция, разработка...)

📁 Комплексные услуги
└── 2 услуги
```

## 🎯 Критерии готовности MVP

### Pre-MVP готов когда:
- [ ] Можно просматривать каталог услуг
- [ ] Работает авторизация с ролями
- [ ] Можно редактировать услуги (TipTap)
- [ ] Видны связи услуг с продуктами
- [ ] Работает экспорт в PDF/Word

### Hi-Fi MVP готов когда:
- [ ] Данные в PostgreSQL
- [ ] Полный CRUD всех сущностей
- [ ] Работает workflow согласования
- [ ] Дашборды с аналитикой
- [ ] Матрица компетенций
