# Тип отчета «Биллинг»

## Назначение

Отчёт **«Биллинг»** содержит данные об оценке, плане и факте выручки по кассовому методу:

- **Оценка** — данные вкладки «Выручка» — «Оценка биллинга» в карточке проекта
- **План** — сохранённые базовые показатели проекта (из мастер-версии)
- **Факт** — данные из счетов

### Права доступа

Для доступа к данным отчёта требуется набор прав для роли **Управление проектами** с правом доступа **«Просмотр»** для гранулы **«Финансовые сведения»**.

Право доступа **«Просмотр»** для гранулы **«Финансовые сведения»** дают также наборы прав для ролей **Управление биллингом** и **Управление финансами**.

## Описание полей

| Поле | Код | Описание |
|------|-----|---------|
| Дата | Date | Дата (оценки, плана или факта) |
| Проект | ProjectId | Аналитика **Проект** |
| Менеджер проекта | ProjectManagerId | Аналитика **Пользователь**. Сотрудник, назначенный менеджером проекта |
| Организация | ClientId | Аналитика **Организация** |
| Программа | ProgramId | Аналитика **Программа** |
| Этап | FirstLevelTaskId | Аналитика **Этап** (работа проекта) |
| План Сумма | PlannedAmount | Данные по биллингу из мастер-версий проектов |
| Оценка Сумма | EstimatedAmount | Данные, указанные на вкладке **Оценка биллинга** карточки проекта |
| Факт Сумма | ActualAmount | Данные по счетам |
| Тип | LineTypeId | Синтетическая аналитика: **«Счёт»** или **«Оплата»** |

**Тип «Счёт»** указывает на использование даты счёта. Для оценки — поле «Дата счёта», для факта — свойство счёта «Выпущено».

**Тип «Оплата»** указывает на использование даты оплаты счёта. Для оценки — поле «Дата оплаты», для факта — свойство счёта «Оплачено».

## Примеры

По проекту "Аудит компании Х" базовые показатели содержали:

| Дата счёта | Дата оплаты | Сумма |
|------------|-------------|-------|
| 01.07.2023 | 15.07.2023 | 100 000 |

Текущая оценка сместилась:

| Дата счёта | Дата оплаты | Сумма |
|------------|-------------|-------|
| 01.07.2023 | 01.08.2023 | 100 000 |

По факту выставили счёт:

| Дата счёта | Дата оплаты | Сумма |
|------------|-------------|-------|
| 15.07.2023 | Не оплачен | 100 000 |

Отчёт по биллингу:

| Тип | Месяц | Оценка Сумма | План Сумма | Факт Сумма |
|-----|-------|--------------|-----------|-----------|
| Счёт | Июль 2023 | 100 000 | 100 000 | 100 000 |
| Оплата | Июль 2023 | 100 000 | | |
| Оплата | Август 2023 | 100 000 | | |

При фильтрации с типом **«Оплата»**:

| Месяц | Оценка Сумма | План Сумма | Факт Сумма |
|-------|--------------|-----------|-----------|
| Июль 2023 | 100 000 | | |
| Август 2023 | 100 000 | | |

**Интерпретация:** планировали получить деньги в июле, но сдвиг на август, и по факту оплата не поступила.
