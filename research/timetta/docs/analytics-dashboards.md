# Использование панелей мониторинга

## Общая информация

**Панель мониторинга** представляет собой комплексный отчёт, объединяющий разные типы виджетов: диаграммы, индикаторы и таблицы данных.

Панель выполняет запрос данных от имени её владельца с учётом его прав доступа. Это означает, что доступ к панели предоставляет доступ к используемым для визуализации данным.

### Панель и отчёт: различия

- Панель выполняет запрос данных от имени владельца, отчёт — от имени текущего пользователя
- Публикация панели эквивалентна публикации данных, в то время как общий отчёт не предполагает публикацию данных
- Отчёт служит способом доступа к детальной информации по узкой области в реальном времени, панель — быстрый комплексный отчёт с показателями и визуализацией с некоторым временным лагом

## Создание панели

Панели создаются в области **Аналитика**. Необходимо перейти в список всех панелей и нажать кнопку **Создать панель** на панели действий.

В открывшемся окне требуется заполнить поле **Наименование** и (по желанию) другие сведения о панели, включая права доступа.

## Настройка виджетов

Только владелец панели может добавлять и настраивать виджеты.

Панель состоит из **виджетов** — информационных блоков, каждый из которых представляет небольшой отчёт со своим набором данных и способом визуализации.

### Процесс создания и настройки виджета:

1. Нажать кнопку **Создать виджет**
2. Навести курсор на созданный виджет и нажать **Редактировать**
3. На вкладке **Главное** изменить название и выбрать тип виджета
4. На вкладке **Данные** выбрать тип отчёта, грейд и до пяти значений
5. На вкладке **Фильтр** указать фильтры для отображаемых данных (если требуется)
6. При использовании повременных отчётов на вкладке **Период** указать временной диапазон
7. Нажать **Сохранить**

Положение и размер виджетов на панели настраиваются путём перемещения. Для удаления виджета используется кнопка **Удалить**.

## Кэширование и обновление данных

Данные панели кэшируются. При открытии проверяется время обновления: если данные устарели, выполняется обновление, иначе возвращаются кэшированные данные.

Для немедленного обновления данных следует нажать кнопку **Обновить**. Эта кнопка доступна только владельцу панели.
