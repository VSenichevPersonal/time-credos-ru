# Шаблоны таймшитов

## Назначение

Шаблоны таймшитов — это компонента для настройки внешнего вида и правил формирования таймшитов. Каждому пользователю назначается шаблон через вкладку "Таймшиты" в его профиле. Эта функция предназначена для опытных администраторов и требует соответствующих прав доступа.

## Создание шаблона

Для создания нового шаблона:
1. Перейти в компоненту Шаблоны таймшитов
2. Нажать кнопку "Создать шаблон"
3. Ввести наименование и подтвердить создание

## Основные разделы карточки

### Главное
- **Наименование** — название шаблона
- **Сущность активна** — включение/отключение использования
- **Шаблон по умолчанию** — автоматическое назначение новым пользователям

### Автоматизация
- Автоматическое создание таймшитов при наступлении периода
- Удаление пустых строк при отправке на согласование

### Согласование
- Лаг для вычисления срока отправки (в рабочих днях)
- Автоматическое согласование при наличии только согласованных отсутствий
- Автоматическое согласование пустых таймшитов

### Интерфейс
Опции видимости элементов в таймшите:
- Код оплаты
- Вид работ
- Роль
- Тариф
- Клиент
- Центр затрат

### Правила валидации
Критерии проверки таймшита при отправке на согласование. Проверка происходит обязательно перед отправкой.

### Дополнительные поля
Пользовательские поля для ячеек или строк таймшита, созданные предварительно в соответствующей компоненте.

### Допустимые состояния проектов
Ограничение доступных проектов по их статусам при заполнении таймшита.
