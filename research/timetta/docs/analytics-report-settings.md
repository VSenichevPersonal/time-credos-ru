# Настройка отчёта | Timetta

Для настройки отчёта необходимо перейти в карточку отчёта и нажать кнопку **Настройка** на панели действий.

## Представление

Представление отчёта определяет источник данных, набор отображаемых полей и их группировку. Форма настройки включает:

1. **Тип отчёта** — определяет набор доступных данных
2. **Применить шаблон** — выбор готовых шаблонов представления
3. **Фильтр колонок** — быстрый поиск необходимых полей
4. **Набор данных** — определяется типом источника
5. **Строки** — поля, используемые в качестве аналитик
6. **Значения** — поля, используемые в качестве значений
7. **Группы строк** — поля для группировки строк
8. **Группы значений** — поля для группировки значений

## Применение шаблона

Шаблон отчёта создается администратором и позволяет заполнить представление автоматически. Для применения: выберите тип отчёта, нажмите кнопку **Применить шаблон** и выберите доступный шаблон.

## Выбор полей

Добавьте поля установкой флага или перетаскиванием в нужную область. Удалите поле снятием флага. Отредактируйте заголовок кликом на кнопку редактирования. Для полей в разделе Значения выбирайте вид итогов: Кол-во, Сумма, Среднее, Макс., Мин. Порядок отображения изменяется перетаскиванием.

## Общий фильтр

*Доступен только в общих отчётах.* Позволяет ограничить набор данных общего отчёта. Рядовые пользователи не имеют доступа к его настройке, но могут добавить собственный пользовательский фильтр.

## Свойства

*Доступны только в общих отчётах и шаблонах.* Позволяют изменить наименование, описание и признак активности отчёта.
