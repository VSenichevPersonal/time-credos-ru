# Заявка на отсутствие

Обновлено: 29.07.2025

## Жизненный цикл заявки на отсутствие

Жизненный цикл по умолчанию:

Жизненный цикл Заявки на отсутствие можно настраивать с помощью компоненты Жизненные циклы. Подробнее — Жизненные циклы.

## Права на переходы

Права доступа позволяют указать исполнителей для каждого перехода. В качестве исполнителей можно указывать:

- Роль
- Группу пользователей
- Конкретных пользователей
- Наборы прав

Для жизненного цикла заявки на отсутствие поддерживаются следующие роли:

| Роль | Описание |
|------|----------|
| Все | Любой пользователь системы |
| Автор | Пользователь — автор документа |
| Руководитель автора | Пользователь, указанный в качестве непосредственного руководителя в карточке пользователя автора |
| Со-менеджеры программы | Пользователи, указанные в качестве Со-менеджеров программ |
| Со-менеджеры проекта | Пользователи, указанные в качестве Со-менеджеров проекта |

## Системные состояния

Системные состояния запускают внутри системы другие процессы:

| Состояние | Описание |
|-----------|----------|
| Черновик | Начальное состояние заявки на отсутствие, допускающий её редактирование при переходе в другое состояние — происходит валидация заполненности полей |
| Согласовано | При переходе заявки на отсутствие в состояние «Согласовано»: заявка блокируется от изменений; создаются записи с типом «Списание» в компоненте Баланс отсутствий |
