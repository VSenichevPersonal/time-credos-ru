# Расписания | Timetta

## Назначение

Справочник содержит список расписаний — графиков рабочего времени сотрудников. "Расписание определяет продолжительность каждого рабочего дня для сотрудника (его «ёмкость»)."

Требуется роль администратора с активной гранулой для управления расписаниями.

## Использование

Расписания применяются в следующих компонентах:

- **Пользователи**: каждому сотруднику назначается свое расписание
- **Проекты**: каждому проекту устанавливается график выходных

Функции расписаний:

- Расчет нормативов рабочего времени и ёмкости сотрудников
- Отображение выходных дней в таймшитах, задачах и ресурсных планах
- Применение правил валидации таймшитов

## Создание расписания

"При развёртывании системы автоматически создается стандартное расписание — «40 часов в неделю»" с 8 рабочими часами с понедельника по пятницу.

Для добавления расписания:

1. Перейти в раздел **Настройки** → **Расписания**
2. Нажать кнопку **Создать расписание**
3. Указать наименование и создать

## Настройка расписания

Существуют два типа расписаний:

**Недельное расписание** — длительность 7 дней. Позволяет задать продолжительность каждого дня (0-24 часа, включая дробные значения).

**Произвольное расписание** — основывается на дате отсчета; используется для графиков, отличных от стандартного недельного (например, "сутки через трое").

## Исключения

Праздники, сокращенные дни и другие исключения настраиваются в справочнике "Исключения расписаний".

## Расписание по умолчанию

Одна запись должна быть назначена расписанием по умолчанию. Оно автоматически присваивается новым пользователям и используется для расчета FTE (full-time equivalent).
