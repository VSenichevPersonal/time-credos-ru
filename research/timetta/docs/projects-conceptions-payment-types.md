# Типы учёта проекта

## Общие сведения

**Тип учёта** проекта определяет способ его учёта и влияет на расчёт выручки, утилизации и биллинга.

Каждому проекту можно назначить только один тип учёта:

1. **Неоплачиваемый** — для внутренних, инвестиционных, административных и других проектов, которые не предполагают оплаты клиентом. У такого проекта есть затраты, но нет выручки.
2. **Фиксированная стоимость** — для проектов с заранее определённой выручкой. Выручка планируется на этапе планирования.
3. **Время и затраты** — для проектов с повременной оплатой. Предполагает выбор тарифов (ставок биллинга), по которым будет формироваться выручка к оплате клиентом.

## Влияние на утилизацию

**Утилизация** — это показатель эффективности сотрудника или подразделения за определённый период (неделя, месяц, квартал).

Формула для расчёта утилизации:

`Утилизация = Оплачиваемые (клиентом) часы за период / Все учтённые часы за период`

### Оплачиваемые часы

Часы считаются оплачиваемыми в двух случаях:

* если они списаны на проекты с типом учёта **«Фиксированная стоимость»**;
* если они списаны на проекты с типом учёта **«Время и затраты»** и в строке таймшита выбран тариф проектов с ненулевой стоимостью.

## Особенности смены типа учёта

В ходе реализации проекта может потребоваться сменить его тип учёта.

**Внимание:** Изменение типа учёта приводит к пересчёту данных задним числом, включая уже утверждённые таймшиты. Выставленные счета останутся без изменения, что по факту является искажением. Для выполнения операции требуются отдельные права через гранулу **«Смена типа биллинга»**.

| Исходный тип учёта | Новый тип учёта | Результат |
|---|---|---|
| Неоплачиваемый | Фиксированная стоимость | Все часы по проекту становятся оплачиваемыми и учитываются в утилизации. |
| Неоплачиваемый | Время и затраты | Часы считаются неоплачиваемыми, пока для них в таймшите не будет выбран тариф. |
| Время и затраты | Неоплачиваемый | Все часы по проекту становятся неоплачиваемыми и не учитываются в утилизации. Все тарифы проекта удаляются, а выбранные тарифы в строках таймшитов — очищаются. |
| Время и затраты | Фиксированная стоимость | Часы остаются оплачиваемыми. |
| Фиксированная стоимость | Неоплачиваемый | Все часы по проекту становятся неоплачиваемыми и не учитываются в утилизации. Все тарифы проекта и выбранные тарифы в строках таймшитов удаляются. |
| Фиксированная стоимость | Время и затраты | Часы считаются неоплачиваемыми, пока для них в таймшите не будет выбран тариф. |
