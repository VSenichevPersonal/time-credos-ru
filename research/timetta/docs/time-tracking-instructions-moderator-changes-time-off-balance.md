# Внесение корректировок в баланс отсутствий

## Обзор

Изменения в балансе отсутствий выполняются через компонент **Баланс отсутствий**. Для этой операции требуется набор прав роли "Управление командой" с правами просмотра и редактирования для гранулы "Начисления и списания отсутствий".

Существует два способа корректировки баланса:
- Добавить новую запись
- Изменить существующую запись

## Добавление новой записи в баланс отсутствий

Процесс создания новой записи:

1. Нажать кнопку **Создать запись**
2. Заполнить необходимые поля (описание доступно в документации "Расчёт баланса отсутствий")
3. Нажать **Создать**

После выполнения этих шагов "запись создана — нужные изменения успешно внесены в баланс отсутствий".

## Изменить имеющуюся запись в балансе отсутствий

Для редактирования существующей записи:

1. Открыть нужную запись, нажав на её номер
2. Внести изменения в поле **Значение** (увеличить или уменьшить значение)
3. Нажать **Сохранить**

После сохранения "запись скорректирована — нужные изменения успешно внесены в баланс отсутствий".
