# Выручка проекта

## Обзор

Вкладка карточки проекта **Выручка** предназначена для оценки и анализа доходной части проекта. Она позволяет:

- вводить оценочные данные;
- анализировать финансовые показатели, включая плановые, фактические и прогнозные значения.

Вкладка упрощает актуализацию оценки и контроль финансовых показателей проекта. Благодаря отображению сведений о выручке, счетах и оплате в одном интерфейсе можно составить комплексное представление о доходах проекта.

## Права доступа

Для доступа к вкладке требуется набор прав для роли **Управление проектами** с хотя бы одной активной гранулой из перечисленных: «Оценка выручки», «Оценка биллинга», «Финансовые сведения».

## Компоненты интерфейса

1. **Переключатель представлений** — для изменения фокуса между календарём, оценкой выручки и оценкой биллинга.
2. **Навигация по периодам** — перемещение вперёд и назад по календарю.
3. **Быстрый переход** на конкретную дату.
4. **Масштабирование представления** — для выбора группировки: по дням, неделям или месяцам.
5. **Мастер выручки** — для расчёта выручки по проекту в зависимости от выбранного метода.
6. **Шкала времени**.
7. **Опции отображения данных** по типам показателей — **План**, **Оценка**, **Факт**, **Прогноз**.
8. **Показатели проекта** (**Выручка**, **Счета** и **Оплата**) и типы показателей.
9. **Работы проекта**, по которым есть данные.

## Примечание

Строки «Прочий факт» и «Прочий план» появляются, когда в проекте есть скрытые данные по удалённым из проекта работам либо по работам из других версий проекта. Для отображения скрытых данных необходимо нажать на иконку глаза в строке прочих показателей.
