# Заполнение таймшита

## Добавление строк в таймшит

Для начала заполнения таймшита требуется добавить новую пустую строку с помощью кнопки **Добавить строку** над таблицей.

Если предыдущие периоды содержали похожие задачи, можно скопировать список работ через ссылку **Скопируйте** или кнопку **Копировать строки из предыдущего**.

Дополнительно можно копировать выделенные строки в буфер обмена кнопкой **Копировать** и вставлять их через иконку **Вставить строки из буфера**.

Кнопка **Создать строки по ресурсному плану** автоматически формирует список работ на основе ресурсных планов проектов, добавляя те задачи, по которым запланированы трудозатраты на текущего пользователя в текущем периоде.

## Выбор работ

В каждой новой строке необходимо выбрать клиента, проект и конкретную работу для списания времени.

## Дополнительные аналитики

Таймшит поддерживает указание дополнительных классификаторов:

- **Код оплаты** — классификатор для расчета почасовой оплаты или премий
- **Вид работ** — дополнительный классификатор из фиксированного списка
- **Роль** — роль исполнителя в проекте, выбирается из ролей, заданных в карточке проекта. Заполняется автоматически при одной роли
- **Центр затрат** — центр для списания фактической себестоимости. Заполняется автоматически при единственном центре затрат
- **Тариф** — тариф участия в проекте, выбирается из списка. Заполняется автоматически при одном тарифе

Администратор может отключить отображение некоторых аналитик.

## Внесение отметок и комментариев

Продолжительность работ указывается по каждой строке на каждый день. К отметкам можно добавлять комментарии через интерфейс таймшита.

Ввод возможен несколькими способами: популярные значения через кнопки, произвольный ввод с клавиатуры или использование счётчика времени (черный треугольник справа от популярных значений).

Продолжительность можно указывать в часах, минутах или процентах от расписания пользователя в зависимости от настройки "Способ ввода времени" в личных настройках.
