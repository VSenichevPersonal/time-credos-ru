# Вычисляемые поля

Обновлено: 11.12.2023

## Назначение

Вычисляемое поле аналитики — это поле источника данных, значение которого вычисляется при построении отчета или формировании панели мониторинга.

Поля настраиваются в области Аналитика, в списке [Вычисляемые поля](https://app.timetta.com/calculated-fields).

## Создание поля

Перейдите в список вычисляемых полей и нажмите кнопку **Создать поле**.

Выберите **Тип отчета**, для которого создается поле, заполните **Наименование** поля и нажмите **Создать**.

## Настройка поля

### Главное

- **Наименование** — внутреннее наименование поля.
- **Тип** — тип поля. Определяет форматирование значений.
- **Сущность активна** — признак активности поля. Неактивные поля остаются в списке, но недоступны для выбора при настройке отчета.

### Заголовок

Определяет заголовок поля в источниках данных с учетом языка пользователя.

### Выражение

Задает выражение для вычисления значения поля. Подробнее — [Выражения вычисляемых полей](https://timetta.com/ru/docs/analytics/expression-of-calculated-fields).

## Использование вычисляемых полей

Вычисляемые поля отображаются в соответствующем отчете в группе «Вычисления».

> **Примечание**
>
> Конфигурация источников данных кэшируется в браузере. После создания вычисляемого поля нужно обновить страницу браузера для отображения нового поля в наборе полей отчета.
