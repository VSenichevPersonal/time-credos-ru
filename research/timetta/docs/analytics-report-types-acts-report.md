# Тип отчёта «Акты»

## Назначение

Этот отчёт служит целью вывода реестра актов. Он содержит информацию об актах и характеристики связанных сущностей — проекта, клиента и программы.

Для работы с данным отчётом требуются права доступа уровня "Просмотр" гранулы "Акты" в наборе прав ролей "Управление проектами" или "Управление финансами".

## Описание полей

Отчёт включает следующие поля данных:

| Поле | Код | Описание |
|------|------|----------|
| Акт | ActOfAcceptanceId | Аналитика Акт |
| Проект | ProjectId | Аналитика Проект |
| Организация | OrganizationId | Аналитика Организация |
| Программа | ProgramId | Аналитика Программа |
| Сумма | Amount | Общая сумма по всем строкам в акте |

Этот отчёт позволяет пользователям получить полный "реестр актов" с агрегированными финансовыми данными и связанной информацией о проектах и организациях.
