# Ресурсный разрыв

## Определение

**Ресурсный разрыв (Resource Gap)** представляет собой метрику, отражающую разницу между требуемым объемом ресурсов и их фактической доступностью за конкретный период времени.

Основная формула: `Resource Gap = Demand – Capacity`

Интерпретация результатов:
- Значение > 0 указывает на **дефицит ресурсов**
- Значение < 0 свидетельствует об **избытке ресурсов**
- Значение = 0 означает **баланс**

В процентном выражении: `Resource Gap (%) = (Demand − Capacity) / Capacity × 100%`

Рекомендуемый диапазон нормы составляет от –5% до +5% на уровне организации и до ±10% для отдельных ролей.

## Понимание ресурсного пула

Система различает два типа ресурсов:

1. **Именные ресурсы** — конкретные сотрудники с учетом их календарей, отпусков и фактической загрузки
2. **Универсальные ресурсы** — типовые профили или роли (например, "Бизнес-аналитик"), используемые на этапе предварительного планирования

## Основные показатели оценки

- **Ёмкость (Capacity)** — доступный объем рабочего времени с учетом расписания и выходных
- **Потребность (Demand)** — суммарный объем запрошенного времени на проектные активности
- **Использование (Utilization)** — степень задействованности доступного времени
- **Ресурсный разрыв** — разница между емкостью и потребностью

## Нормативные диапазоны

| Уровень анализа | Норма | Интерпретация |
|---|---|---|
| На уровне компании | –5% … +5% | Баланс (идеальное состояние) |
| На уровне роли | –10% … +10% | Допустимое колебание |
| На уровне проекта | –15% … +15% | Норма для динамичных сред |
| Дефицит > +10% | — | Риск невыполнения планов |
| Избыток < –10% | — | Низкая загрузка |

## Цветовая шкала в Timetta

- Синий (< –10%): избыток ресурсов
- Зеленый (–10% … +10%): баланс
- Оранжевый (+10% … +20%): управляемый дефицит
- Красный (> +20%): критический дефицит
