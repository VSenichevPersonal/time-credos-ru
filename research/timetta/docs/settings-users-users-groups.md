# Группы пользователей | Timetta

Обновлено: 21.08.2025

## Назначение

Компонента предназначена для создания, настройки и управления группами пользователей.

> **Примечание**
>
> Настройка групп пользователей доступна в компоненте [**Группы пользователей**](https://app.timetta.com/settings/groups), если пользователю назначен набор прав для роли **`Администратор`** с активной гранулой «Группы пользователей».

## Использование

Способы применения групп пользователей:

- Для назначения групповых прав доступа. На уровне группы можно задать наборы прав, тогда все пользователи, добавленные в группу, получат соответствующие права.
- Группу пользователей можно назначить в качестве инициатора [воркфлоу](https://timetta.com/ru/docs/settings/config/workflow).
- Группе пользователей можно назначить права на принудительную смену стадий [жизненного цикла](https://timetta.com/ru/docs/settings/config/life-cycle) сущностей.

## Создание группы пользователей

Чтобы создать группу пользователей, нужно:

1. Перейти к компоненте [**Группы пользователей**](https://app.timetta.com/settings/groups).
2. На панели действий нажать на кнопку **Создать группу**.
3. Указать наименование новой группы и нажать на кнопку **Создать**.

## Управление составом группы

Список включённых в группу пользователей отображается на вкладке **Пользователи** в карточке группы.

Для добавления пользователей в группу нужно:

1. В карточке группы перейти на вкладку **Пользователи**.
2. На панели действий нажать на кнопку **Добавить пользователей**.
3. Выбрать пользователей, которых нужно добавить в группу, и нажать на кнопку **Добавить**.

Для исключения одного или нескольких пользователей из группы нужно:

1. В карточке группы перейти на вкладку **Пользователи**.
2. Выбрать одного или нескольких пользователей в списке и нажать на кнопку **Исключить**:

![Исключение пользователей из группы](/entity-assets/fde5c294-0078-459a-b05f-316c5f34aaad/users-group-new.png)

Пользователь может одновременно входить в состав нескольких групп. Все группы, в которые входит пользователь, перечислены в его карточке на вкладке **Группы**. Также на этой вкладке можно добавлять или исключать пользователя из групп.

## Наборы прав

На вкладке **Наборы прав** в карточке группы указываются наборы прав, назначенные для группы пользователей.

Итоговые права пользователя включают все назначенные ему индивидуальные и групповые наборы прав (объединяются индивидуальные права и права, назначенные через группы).
