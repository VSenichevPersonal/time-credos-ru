# Тип отчёта «Пользователи»

Отчёт «Пользователи» содержит информацию о пользователях и фактических данных.

## Права доступа

Для доступа к данным отчёта требуется набор прав для роли **Управление командой** с правами доступа «Просмотр» для гранулы «Профиль пользователя».

Для доступа к финансовым данным отчёта дополнительно требуется гранула **«Финансовые сведения»** из того же набора прав.

## Описание полей

| Поле | Код | Описание |
|------|-----|----------|
| Пользователь | UserId | Аналитика Пользователь |
| Подразделение | DepartmentId | Аналитика Подразделение |
| Руководитель | SupervisorId | Аналитика Пользователь. Сотрудник, назначенный руководителем выбранного пользователя в настройках |
| Группы | Groups | Группы, в которые входят пользователи |
| Лицензии | Licenses | Продукты, назначенные пользователю |
| Ресурсный пул | ResourcePoolId | Аналитика Ресурсный пул. Ресурсный пул, в который входит пользователь |
| Менеджер ресурсного пула | ResourcePoolManagerId | Аналитика Пользователь. Пользователь, назначенный менеджером ресурсного пула в настройках |
| Уровень | LevelId | Аналитика Уровень. Должностной уровень сотрудника |
| Локация | LocationId | Аналитика Локация. Местонахождение сотрудника |
| Грейд | GradeId | Аналитика Грейд. Должностной суб-уровень пользователя |
| Себес. валюта | CurrentCostCurrencyId | Валюта текущей ставки себестоимости |
| Себес. ставка | CurrentCostRate | Ставка себестоимости пользователя (текущая) |
| Себес. ставка БВ | CurrentCostRateBC | Ставка себестоимости пользователя (текущая) в базовой валюте |
| Себестоимость | Cost | Себестоимость по проектам и по согласованным отсутствиям, входящим в существующий таймшит. Появляется в момент согласования таймшита, в который входят часы |
| Себестоимость БВ | CostBC | Себестоимость в базовой валюте по проектам и по согласованным отсутствиям, входящим в существующий таймшит. Появляется в момент согласования таймшита, в который входят часы |
| Себестоимость Норм. | CostNormalized | Нормализованная себестоимость по проектам и по согласованным отсутствиям, входящим в существующий таймшит. Появляется в момент согласования таймшита, в который входят часы |
| Себестоимость Норм. БВ | CostNormalizedBC | Нормализованная себестоимость в базовой валюте по проектам и по согласованным отсутствиям, входящим в существующий таймшит. Появляется в момент согласования таймшита, в который входят часы |
| Юридическое лицо | LegalEntityId | Аналитика Юридическое лицо. Юридическое лицо, к которому относится центр затрат |
| Часы Всего | HoursTotal | Часы по проектам из согласованных таймшитов + Часы отсутствий из согласованных заявок на отсутствия |
| Часы Всего за Пер. | HoursTotalOverPeriod | Сумма часов, списанных пользователем на проекты и отсутствия (в таймшитах с любым состоянием) за период. Подробнее — Особенности полей «Расчёт времени за период» |
| Часы Оплачиваемые | HoursBillable | Часы из согласованных таймшитов по оплачиваемым проектам (с типом оплаты «Фиксированная стоимость» или «Время и затраты») |
| Часы Неоплачиваемые | HoursNonBillable | Часы из согласованных таймшитов по проектам с типом «Неоплачиваемый» + Часы из согласованных заявок на отсутствие |
| Часы Проекты | HoursProjects | Часы по всем проектам из согласованных таймшитов |
| Часы Отсутствия | HoursTimeOffs | Часы из согласованных заявок на отсутствие |
| Часы Расписание за пер. | HoursScheduleOverPeriod | Сумма часов по расписанию за выбранный период. Для расчёта используется расписание, установленное в карточке пользователя |
| Часы FTE за пер. | HoursFteScheduleOverPeriod | Сумма часов по расписанию за выбранный период. Для расчёта используется расписание со свойством «По умолчанию» |
| Часы Норм. | HoursNormalized | Нормализованные часы по проектам (из согласованных таймшитов) + Часы из согласованных заявок на отсутствие, входящих в таймшит |
| Бронь Часы за пер. | BookedHoursOverPeriod | Сумма забронированных на пользователя Hard-часов из календаря бронирования |
| План Часы за пер. | PlannedHoursOverPeriod | Сумма часов, запланированных на пользователя в ресурсных планах мастер-версий проектов |
| Выручка T&M | Revenue | Сумма часов трудозатрат пользователя из согласованных таймшитов, умноженных на ставки биллинга ролей |
