# Учётные статьи

Обновлено: 28.08.2025

## Статьи

**Учётные статьи** — это классификатор финансовых операций, используемый для группировки и анализа доходов и затрат по проектам.

Учётные статьи применяются:

* при планировании;
* при учёте;
* при анализе финансовых показателей.

Они позволяют формировать отчётность с детализацией по типам доходов и затрат.

> Классификатор доступен в компоненте **Учётные статьи**, если пользователю назначена роль _Администратор_ с активной гранулой **Учётные статьи**.

В системе есть системные статьи. Их нельзя удалить или изменить:

| Учётная статья | Тип | Назначение |
|---|---|---|
| **Выручка** | Доход | Основная статья для учёта доходов проекта |
| **Себестоимость труда** | Затраты | Стоимость рабочего времени сотрудников, списываемого на проект |
| **Себестоимость труда подрядчиков** | Затраты | Стоимость работ внешних подрядчиков |
| **Стоимость отсутствий** | Затраты | Учёт стоимости оплачиваемых отсутствий (отпуск, больничный и другие) |
| **Корпоративные налоги** | Затраты | Учёт расходов по «Корпоративному налогу» проекта |
| **Стоимость капитала** | Затраты | Финансовые издержки, связанные со стоимостью капитала или внутренним финансированием |

Организация может создавать дополнительные учётные статьи, например:

* командировочные расходы;
* лицензии программного обеспечения;
* оборудование;
* обучение и сертификация;
* прочие операционные расходы;
* консалтинговые услуги;
* разработка;
* внедрение системы;
* техническая поддержка;
* сервисный контракт / подписка;
* дополнительные услуги.

## Использование

Классификатор используется в следующих компонентах:

* оценка затрат и выручки проекта (обязательно);
* оценка биллинга и расчёт со субподрядчиками (опционально);
* проводки;
* акты;
* заявки на затраты;
* финансовые отчёты (аналитика для анализа).

## Создание учётной статьи

Чтобы создать учётную статью:

1. Перейдите в компонент **Учётные статьи**.
2. На панели действий нажмите кнопку **Создать учётную статью**.
3. Укажите **Наименование**, **Тип**, **Код** и нажмите **Создать**.

## Настройка учётной статьи

Свойства карточки:

* **Наименование** — название учётной статьи;
* **Тип** — определяет принадлежность статьи к **Выручке** или **Затратам**;
* **Код** — уникальный код статьи;
* **Описание** — краткое описание назначения статьи;
* **Учитывать в расчёте баланса** — позволяет учитывать статью при расчёте баланса базового финансового отчёта проекта. Если параметр отключён, статья учитывается только в расширенном финансовом отчёте;
* **Сущность активна** — признак активности записи. Неактивные учётные статьи недоступны для выбора в новых заявках на затраты.
