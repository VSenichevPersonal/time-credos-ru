# Массовые операции с таймшитами и заявками на отсутствия

## Обзор

С помощью массовых операций можно выполнять однотипные действия над двумя и более сущностями. Система позволяет массово согласовывать несколько таймшитов или заявок на отсутствие, что экономит время при работе с большим количеством данных.

## Выполнение массовой операции

Чтобы применить массовую операцию к таймшитам или заявкам на отсутствие:

1. Перейти к списку сущностей
2. Выбрать нужные сущности, отметив их чекбоксом в первом столбце
3. Отметить чекбокс в строке заголовков таблицы для выбора всех сущностей
4. Нажать на кнопку **Изменить состояние**
5. Выбрать операцию из выпадающего списка

## Особенности выполнения

- После выбора операции открывается диалоговое окно, показывающее количество выбранных сущностей
- Некоторые операции требуют указания комментария
- Для большого количества сущностей операция может занять продолжительное время
- Операция применится только к сущностям, для которых допустим переход в соответствующее состояние
- Нельзя применять операции к сущностям в разных состояниях

Примечание: если правила валидации не позволяют изменить состояние некоторых сущностей, то операция не сработает для них.

Доступные операции включают:
- Согласовать
- Редактировать (переводит в состояние «Черновик»)
- Переоткрыть (открыть уже согласованную сущность)

Список доступных операций определяется настройками жизненного цикла системы.
