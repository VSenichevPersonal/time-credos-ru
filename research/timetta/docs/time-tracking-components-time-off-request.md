# Заявка на отсутствие

## Назначение

Заявки на отсутствия предназначены для учёта отпусков, больничных и прочих видов отсутствий. Система поддерживает настройку процесса согласования через воркфлоу.

### Права доступа

- Доступ к личным заявкам требует набора прав для роли "Пользователь" с активной гранулой "Заявки на отсутствия"
- Доступ к заявкам других пользователей требует прав для роли "Управление командой" с активной гранулой "Заявки на отсутствие"

## Использование

Заявки на отсутствие интегрируются с несколькими компонентами системы:

- **График отсутствий** — заявки отражаются в графике отсутствий
- **Баланс отсутствий** — согласованные заявки влияют на оставшийся баланс сотрудника
- **Таймшиты** — заявки включаются в таймшиты
- **Бронирование** — периоды отсутствий уменьшают доступную ёмкость сотрудников

## Обзор

Карточка заявки содержит следующие элементы:

- **Тип отсутствия** — классификатор, выбираемый из доступного списка
- **Период** — даты начала и окончания отсутствия с опцией редактирования
- **Сведения об отсутствии** — показывает баланс накопленных дней/часов и остаток доступный для списания
- **Описание** — дополнительная информация к заявке
- **Вложение** — возможность прикрепления файлов (заявления, справки)
