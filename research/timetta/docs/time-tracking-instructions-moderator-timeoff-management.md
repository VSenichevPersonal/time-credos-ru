# Управление заявками на отсутствие

## Обзор

Функция **Управление заявками на отсутствие** (ЗнО) позволяет руководителям и менеджерам по кадрам выполнять операции с заявками пользователей. Для доступа требуется набор прав роли «Управление командой» с активированной гранулой для заявок на отсутствие.

## Руководитель

Руководители могут работать только с заявками своих подчиненных в области **Команда**. Доступные действия:

- Создание заявки от имени подчиненного
- Перевод заявки из состояния "На согласовании" в "Черновик" (при наличии соответствующих настроек)
- Удаление заявки (необратимое действие)
- Редактирование заявки в состоянии "Черновик"

### Создание заявки за подчинённого

Процедура:

1. Перейти в область **Команда** и открыть список **Все отсутствия**
2. Нажать кнопку **Создать заявку** на панели действий
3. В поле **Заявитель** указать подчиненного
4. Заполнить остальные поля согласно инструкции по созданию заявки на отсутствие

### Удаление заявки за подчинённого

Удаление доступно только для заявок в состоянии "Черновик":

1. Перейти в область **Команда**, открыть список **Все отсутствия**
2. Найти заявку (использовать фильтры при необходимости)
3. Выбрать одну или несколько заявок
4. Нажать кнопку **Удалить**
5. Если заявка не в состоянии "Черновик", нажать **Переоткрыть** предварительно

**Важно:** удаление необратимо.

## Менеджер по кадрам

Менеджер по кадрам работает с заявками всех пользователей. Доступные действия:

- Создание заявки от имени любого пользователя
- Прекращение согласования (перевод в "Черновик")
- Отмена согласования (перевод из "Согласовано" в "Черновик")
- Удаление заявки
- Редактирование заявки в состоянии "Черновик"
- Принудительное согласование или отклонение

### Отмена согласования заявки

1. Открыть список **Все отсутствия** в области **Команда**
2. Найти заявку (использовать фильтры)
3. Открыть заявку и изменить состояние на "Черновик"

### Принудительное согласование или отклонение

1. Найти заявку в списке **Все отсутствия**
2. На панели действий открыть меню **Изменить состояние**
3. Выбрать **Согласовать** или **Редактировать**

Действие доступно для заявок в состояниях "На согласовании" и "Черновик".

**Важно:** ручное изменение состояния автоматически прерывает воркфлоу после подтверждения.
