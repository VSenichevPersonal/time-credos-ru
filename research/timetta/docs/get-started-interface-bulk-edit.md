# Массовое изменение сущностей

## Алгоритм использования

1. Выберите нужные сущности в списке, отметив их чекбоксами.
2. На панели действий нажмите кнопку **Изменить**.
3. В открывшемся окне выберите свойства, которые хотите обновить, и задайте для них новые значения.
4. Подтвердите действие — изменения применятся ко всем выбранным сущностям одновременно.

## Особенности функции

- массовое изменение доступно только в режиме отображения **Список**; в режиме **Доска** одновременное редактирование нескольких сущностей не поддерживается;
- функция работает для любых представлений в режиме списка — как системных, так и созданных вручную;
- настройки правил валидации и жизненных циклов сохраняются — система не позволит выполнить некорректную операцию;
- изменить можно только те свойства, которые доступны для редактирования в карточке сущности.

## Пример

Если необходимо перевести несколько проектов на другого менеджера, не обязательно открывать карточку каждого проекта и менять руководителя вручную. Достаточно выполнить следующие шаги:

1. Откройте представление **Активные проекты**.
2. Выберите нужные проекты.
3. Нажмите кнопку **Изменить**.
4. В открывшемся окне выберите свойство **Менеджер проекта** и укажите нового сотрудника.
5. Подтвердите действие — все выбранные проекты будут назначены на нового менеджера.
