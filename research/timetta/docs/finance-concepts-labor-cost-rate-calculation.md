# Расчёт ставки себестоимости труда

## Расчёт себестоимости часа

### Расчёт по типу оплаты

Управленческая себестоимость часа предназначена для планирования и оценки фактических затрат проекта.

Для почасовой оплаты: себестоимость соответствует часовой ставке плюс налоги и отчисления.

Для фиксированной оплаты:
- Рассчитываются полные прямые затраты за период
- Учитываются будущие изменения (повышения зарплаты)
- Оценивается ожидаемый объём рабочих часов
- Себестоимость часа = Затраты ÷ Ожидаемые часы

### Тонкие моменты

При расчётах нужно решить:

1. **Период расчёта** — месячный позволяет чаще обновлять ставку, но колебания (премии, бонусы) могут искажать показатели
2. **Объём плановых часов** — можно использовать производственный календарь или индивидуальное расписание сотрудника

### Варианты подходов к точности

- **Простой:** единая средняя ставка для всей компании
- **Продвинутый:** матрица себестоимости по ролям и категориям
- **Точный:** индивидуальный расчёт для каждого сотрудника

## Нормализация себестоимости

Нормализация решает две задачи:
- При переработке зарплата не растёт — применяется коэффициент нормализации (0–1)
- При оплачиваемом отсутствии себестоимость снижается

## Включение косвенных затрат в ставку

Косвенные затраты (отсутствия, некоммерческие проекты, офис) иногда включают в ставку себестоимости.

Международная практика различает:
- **Unloaded Cost Rate** — чистая ставка
- **Loaded Cost Rate** — ставка с накладными (например, коэффициент 1.3)

## Бухгалтерская и управленческая себестоимость

**Бухгалтерская** — только фактическая, за прошедший период на основе документов бухгалтерии.

**Управленческая** нужна для:
- Планирования проектов
- Оценки эффективности команды по исходным ставкам
- Сглаживания колебаний в процессе работы

Обе метрики нужны руководству: управленческая показывает результативность команды, бухгалтерская — реальную финансовую картину.

## Частые ошибки

1. Путаница между себестоимостью часа и ставкой для клиента
2. Требование списывать ровно 8 часов в день (нормализация это решает)
3. Смешивание управленческой и бухгалтерской себестоимости
4. Забывают, что себестоимость есть и у некоммерческих проектов и отсутствий
