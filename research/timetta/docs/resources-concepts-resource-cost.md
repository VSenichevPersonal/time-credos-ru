# Определение себестоимости ресурсов

Обновлено: 22.08.2025

Как правило, на этапе предварительной оценки проекта и формирования калькуляции («прайсинг») команда проекта формируется из универсальных ресурсов.

Затем под каждый универсальный ресурс подбирается конкретный сотрудник.

На этапе калькуляции необходимо:

- Сформировать команду.
- Запланировать часы.
- Получить оценку себестоимости проекта.

## Определение себестоимости для универсальных ресурсов

Для универсальных ресурсов себестоимость определяется по **Матрице себестоимости**, которая настраивается в рамках [Матрицы Ставок](https://timetta.com/ru/docs/settings/finances/rate-matrices).

Ставки в рамках матрицы зависят от [характеристик ресурса](https://timetta.com/ru/docs/resources/concepts/resource-structure#harakteristiki-tipov-resursa). Каждый клиент может самостоятельно определить, какие параметры необходимо учитывать при расчёте себестоимости. Таким образом, матрицу можно адаптировать под индивидуальные потребности.

## Определение себестоимости для сотрудников

Для сотрудников расчёт себестоимости задается индивидуально — на уровне пользователя в Timetta — и рассчитывается согласно заработной плате сотрудника.

Подробнее — [Учёт себестоимости труда](https://timetta.com/ru/docs/finance/concepts/cost-management) для разных типов ресурсов.
