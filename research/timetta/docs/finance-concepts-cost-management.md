# Учёт себестоимости

Обновлено: 28.07.2025

## Себестоимость труда

**Себестоимость труда (Labor Cost)** — затраты на сотрудников, которые участвуют в выполнении проекта. Включает:

- Зарплату
- Зарплатные налоги
- Социальные и страховые отчисления
- Иногда — обучение и другие расходы

## Настройка ставок

Ставки себестоимости можно задать для конкретного сотрудника или для универсального ресурса.

### Для конкретного сотрудника

Для **именованных ресурсов** ставки себестоимости задаются на двух уровнях:

- В карточке пользователя. Это базовая ставка себестоимости, используемая по умолчанию. Чтобы изменить ставку, нужно добавить новое значение с датой начала действия. Подробнее — [Карточка пользователя](https://timetta.com/ru/docs/settings/users/user-settings#sebestoimost).
- На уровне проекта. Такая ставка дополняет сведения из карточки пользователя и имеет приоритет над ними. Позволяет скорректировать стоимость труда конкретного сотрудника в рамках отдельного проекта.

Подробнее о способах расчёта почасовой ставки — [Расчёт ставки себестоимости труда](https://timetta.com/ru/docs/finance/concepts/labor-cost-rate-calculation).

### Для универсального ресурса

Для **универсальных ресурсов** ставки себестоимости задаются через матрицу себестоимости. Она настраивается в рамках матрицы ставок.

Каждый клиент настраивает матрицу индивидуально. Соответственно, способ детализации может быть разным. Например, можно учитывать только уровень или уровень и грейд. Подробнее — [Создание матрицы ставок](https://timetta.com/ru/docs/settings/finances/rate-matrices#sozdanie-matricy-stavok).

## Оценка, планирование и прогнозирование

- Оценка себестоимости вводится в карточке проекта через ресурсный план.
- Мастер-версия проекта фиксирует плановую себестоимость.
- В карточке проекта отображается прогноз. Он равен сумме факта до точки прогнозирования + актуальная оценка до конца проекта.

## Учёт факта

Фактическая себестоимость формируется на основе таймшитов. В проектах отражается себестоимость только по утверждённым таймшитам.

> **Примечание.** Фактическая себестоимость может автоматически нормализоваться — [Методика нормализации себестоимости](https://timetta.com/ru/docs/finance/concepts/cost-normalization-methods).
