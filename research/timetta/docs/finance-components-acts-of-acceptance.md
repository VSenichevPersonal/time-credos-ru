# Акты

Обновлено: 31.07.2025

## Назначение

**Акты** предназначены для учёта фактических данных по выручке в проектах.

**Права доступа**

Для создания и редактирования актов в области [Проекты](https://app.timetta.com/acts-of-acceptance?navigation=projects.actsOfAcceptance) требуется набор прав для роли **Управление проектами** с активной гранулой «Акты».

Для создания и редактирования актов в области [Финансы](https://app.timetta.com/acts-of-acceptance?navigation=finance.actsOfAcceptance) требуется набор прав для роли **Управление финансами** с активной гранулой «Акты».

## Обзор

### Карточка акта

Карточка акта включает следующие поля:

- **Номер** — номер акта (автоматический или введённый вручную).
- **Дата** — дата составления акта. Обязательное для заполнения поле.
- **Дата признания** — дата признания акта, по которой учитывается факт выручки.
- **Описание** — краткое описание акта.

В блок **Вложения** можно добавить любые связанные с актом файлы в разрешённом формате.

Строки таблицы в карточке акта содержат следующие данные:

- **Работа** — этап проекта, к которому относится выручка.
- **Описание** — краткое описание конкретной строки.
- **Сумма** — сумма по строке (в валюте проекта).
- **Обменный курс** — курс конвертации в базовую валюту системы. По умолчанию указывается обменный курс, который актуален на дату акта; при необходимости курс можно скорректировать вручную.
- **Учётная статья** — учётная статья, по которой учитывается факт выручки.
- **Сумма БВ** — сумма по строке в базовой валюте.

Подробнее — [Создание и редактирование актов](https://timetta.com/ru/docs/finance/instructions/act-create&edit).
