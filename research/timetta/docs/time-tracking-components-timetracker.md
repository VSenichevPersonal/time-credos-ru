# Счётчик времени | Timetta

## Назначение

**Счётчик времени** — инструмент для учёта продолжительности работ.

## Создание отметки рабочего времени

Запустить счётчик времени можно с помощью кнопки на верхней панели навигации из любого раздела в системе или из ячейки таймшита.

Каждый запуск счётчика автоматически создает новую отметку времени.

После остановки счётчика пользователь может заполнить форму и сразу внести данные в журнал рабочего времени. Форму можно заполнить позже при редактировании таймшита.

## Заполнение отметки рабочего времени

При нажатии на запущенный счётчик времени откроется форма «Отметки времени». Она позволяет заполнить данные для отправки в таймшит.

В форме «Отметки времени» доступны следующие поля:

- **Задача** — позволяет выбрать задачу из списка доступных пользователю задач.
- **Клиент > Проект > Работа** — позволяет указать конкретную работу проекта для списания времени.
- **Вид работ** — позволяет выбрать тип работ из выпадающего списка.
- **Тариф проекта** — позволяет указать ставку для роли в проекте.
- **Комментарий** — позволяет добавить небольшое пояснение к отметке времени.
- **Красная кнопка** — останавливает счётчик.

## Просмотр и редактирование отметок рабочего времени

Все отметки времени попадают в «Журнал рабочего времени» — специальный раздел таймшита.

Дата отметки времени в журнале — это дата запуска счётчика. Отметки времени без указанных работ, типа работ и тарифа хранятся в отдельной строке журнала.

Для редактирования отметки времени нужно:

1. Выбрать ячейку в журнале рабочего времени.
2. В открывшемся списке нажать на нужную отметку.
3. Заполнить и сохранить форму редактирования отметки рабочего времени.
