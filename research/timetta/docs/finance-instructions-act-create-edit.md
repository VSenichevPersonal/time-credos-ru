# Создание и редактирование актов

## Обзор

Эта документация описывает процесс создания и редактирования актов (актов приемки-передачи) в системе Timetta. Акты могут быть созданы двумя основными способами и включают различные функции для управления.

## Создание акта

### Из областей Проекты и Финансы

Процесс включает следующие шаги:

1. Переход в список актов через области **Проекты** или **Финансы**
2. Нажатие кнопки **Создать акт** на панели действий
3. Выбор проекта и указание номера акта
4. Возможность отключения автоматической нумерации для использования произвольного номера
5. Нажатие кнопки **Создать**

Документ создается в валюте выбранного проекта.

### Из карточки проекта

Альтернативный способ включает:

1. Создание оценки выручки на вкладке **Выручка** проекта
2. Переход в представление «Оценка выручки»
3. Выделение нужной строки оценки
4. Нажатие кнопки **Создать акт**

## Работа со строками акта

Функционал редактирования включает:

- Добавление строк через кнопку **Добавить строку**
- Настройку видимых полей через меню представления
- Редактирование всех свойств строки (обменный курс редактируется только при отличии валюты акта от базовой)
- Удаление строк через контекстное меню

## Лента активности

Система отслеживает все изменения, включая:

- Смену статусов
- Добавление комментариев
- Прикрепление вложений

Комментарии доступны для актов в любых состояниях. Файлы в комментариях собираются на вкладке **Вложения**.

## Признание актов

При переводе акта в состояние "Признано" требуется указать **Дату признания**, которая определяет период, в который попадает фактическая сумма по акту.
