# Общие сведения о Timetta

Обновлено: 29.08.2025

**Timetta (Тиме́тта)** — решение для проектно-ориентированного бизнеса. Оно относится к классу программного обеспечения *Professional Services Automation* (PSA) и *Project Portfolio Management* (PPM).

> **Классификация**
>
> **PSA-система** (от англ. *Professional Services Automation* — автоматизация профессиональных услуг) — это программное обеспечение для управления проектами, ресурсами, финансами и взаимоотношениями с клиентами в компаниях, оказывающих профессиональные услуги.
>
> К таким компаниям относятся, например:
>
> - IT-консалтинг;
> - Юридические фирмы;
> - Маркетинговые агентства;
> - Инжиниринговые организации.
>
> PSA — это специализированная ERP-система, адаптированная для сферы консалтинга и профессиональных услуг.
>
> **PPM-система** (от англ. *Project Portfolio Management*) — это программное обеспечение для управления портфелем проектов компании.
>
> Она помогает:
>
> - Оценивать приоритеты проектов;
> - Распределять ресурсы;
> - Контролировать выполнение стратегических целей.

Решение доступно по модели SaaS из облака Yandex Cloud или в формате On-Premises для развёртывания в локальной ИТ-инфраструктуре.

Клиентская часть оптимизирована для работы в последних и предпоследних версиях браузеров:

- Google Chrome;
- Safari;
- Edge;
- Firefox.
