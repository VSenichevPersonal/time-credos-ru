# Управление счетами

Обновлено: 05.08.2025

## Создание счетов

Создать счёт можно несколькими способами:

- Перейти в область **Биллинг** и открыть [Сводку по биллингу](https://app.timetta.com/billing-summary?navigation=billing.summary).
- Выбрать строку с клиентом и нажать на кнопку **Создать** на панели действий.
- Также счёт можно создать из любого списка счетов (например, **Все счета**) в области **Биллинг**.

Для создания счёта следует заполнить форму:

![Новый счёт](https://timetta.com/entity-assets/5e8212a4-6d35-4595-9a4a-31dc99b00591/new-bill_.png)

- **Клиент** — организация, для которой создаётся счёт. Обязательное для заполнения поле.
- **Проект** — проект, для которого создаётся счёт. Обязательное для заполнения поле.
- **Присвоить автономер** — если отметить чекбокс, будет присвоен следующий номер по порядку. Номера сквозные по всем периодам, клиентам и проектам. Если убрать отметку, появится поле **Номер**.
- **Номер** — номер счёта для ввода вручную.

При создании счёта автоматически выбирается шаблон по умолчанию. Позже его можно изменить, пока счёт находится в состоянии «Черновик».

**Особенности работы с НДС**

Весь учёт в Timetta ведётся без учёта НДС. Если нужно выпустить счёт с НДС, следует включить возможность работы с ним в [Настройках системы](https://app.timetta.com/settings/settings/applications). Далее в шаблоне счёта нужно указать применяемую ставку.

НДС будет добавлен в расчётные значения счёта и печатную форму (документ в формате PDF).

### Создание счёта из проекта

Счёт можно сформировать по конкретному проекту. Для этого нужно:

1. Перейти на вкладку **Выручка** проекта.
2. Переключиться на представление **Оценка биллинга**.
3. Нажать на кнопку **Создать счёт**.

Подробнее — [Оценка выставления счетов и получения оплат](https://timetta.com/ru/docs/projects/instructions/billing-estimate).

## Выставление счетов

Учёт факта выставления и оплаты счёта производится в области [Биллинг](https://app.timetta.com/billing).

Созданный в Timetta счёт считается фактически выставленным при установке состояния «Выпущен» в карточке счёта.

![Выпуск счёта](https://timetta.com/entity-assets/5e8212a4-6d35-4595-9a4a-31dc99b00591/issue-invoice-new.png)

> **Примечание**
> Процесс выставления счетов может быть настроен с помощью [воркфлоу](https://timetta.com/ru/docs/settings/config/workflow).

## Учёт оплаты счетов

При получении оплаты по счёту для него устанавливается состояние «Оплачен».

![Признание оплаты по счёту](https://timetta.com/entity-assets/5e8212a4-6d35-4595-9a4a-31dc99b00591/paid-invoice-new.png)

Эти изменения можно внести вручную. Также они могут быть выполнены автоматически через интеграцию с внешними системами.
