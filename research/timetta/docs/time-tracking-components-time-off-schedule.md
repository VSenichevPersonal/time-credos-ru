# График отсутствий

Обновлено: 26.08.2025

## Назначение

График отсутствий отображает сведения по заявкам на отсутствие в календарном представлении. Инструмент позволяет контролировать пересечения отсутствий между сотрудниками.

**Права доступа**

Для просмотра графика отсутствий требуется набор прав для роли Управление командой с активной гранулой Заявки на отсутствие (как минимум Просмотр).

График отсутствий формируется по всем заявкам на отсутствие и позволяет ресурсным и проектным менеджерам планировать загрузку ресурсов на текущих и будущих проектах.

## Обзор

График отсутствий доступен в компоненте График в области Команда:

1. Переключение между периодами графика.
2. Кнопка «Выбор периода» в календаре.
3. Панель фильтрации. Настройка отображения сотрудников по заданным параметрам.
4. Сводка об отсутствии: основная информация и ссылка на заявку.

Легенда таблицы:

- Штриховкой выделены нерабочие дни пользователя на основе его расписания.
- Буквенное обозначение — код типа отсутствия.
- Цвета ячеек соответствуют цвету состояния заявки на отсутствие, который задан в настройках жизненного цикла. Подробнее — Жизненный цикл.
