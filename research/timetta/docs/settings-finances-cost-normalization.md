# Нормализация себестоимости

## Назначение

Компонента предназначена для создания и настройки правил нормализации себестоимости. Правило, по которому будут нормализоваться часы пользователя, выбирается в карточке пользователя на вкладке «Себестоимость».

В компоненте можно задать правило по умолчанию, которое будет автоматически назначаться пользователю при его создании.

**Примечание:** Правила нормализации себестоимости доступны в компоненте "Норм. себестоимости" если пользователю назначен набор прав для роли **Администратор** с активной гранулой «Нормализация себестоимости».

## Создание правила нормализации

1. Перейдите к компоненте "Норм. себестоимости"
2. На панели действий нажмите кнопку **Создать правило**
3. Укажите Наименование и нажмите кнопку **Создать**

## Настройка нормализации

### Параметры карточки нормализации:

* **Наименование** — наименование правила нормализации
* **Метод нормализации** — метод расчета нормализованных часов:
  * **Равномерно** — единый фактор нормализации для всех проектов вне зависимости от типа оплаты и оплачиваемых отсутствий
  * **Вначале оплачиваемые проекты** — два фактора: приоритетный (для оплачиваемых проектов) и неприоритетный (для неоплачиваемых проектов и отсутствий)
* **Исключаемые типы отсутствий** — список типов отсутствий, снижающих емкость пользователя (обычно неоплачиваемые типы)

**Примечание:** Период нормализации соответствует периоду каждого таймшита. Нормализация часов выполняется при согласовании таймшита, базовая емкость пользователя рассчитывается по расписанию за период таймшита.
