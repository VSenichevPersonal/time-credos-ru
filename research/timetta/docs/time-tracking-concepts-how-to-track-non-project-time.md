# Учёт непроектного времени

Обновлено: 09.08.2025

## Непроектные работы и их виды

Кроме работ, которые входят в конкретный проект (внешний, внутренний и пр.), в компании могут быть непроектные активности и процессы.

Например:

- Административные работы
- Администрирование ИТ
- Взаимодействие с клиентами
- Взаимодействие с коллегами
- Изучение корпоративных технологий и решений
- Изучение новых технологий и решений
- Корпоративные мероприятия
- Обучение и наставничество
- Учёт и планирование рабочего времени
- Простой

Хотя время, потраченное сотрудниками на эти работы, непроектное, его тоже важно учитывать.

## Как вести учёт непроектного времени

Для учёта непроектного времени в Timetta нужно создать проект. Рекомендуется использовать отдельный проект на всю компанию.

Если набор непроектных активностей отличается в каждом подразделении, следует использовать отдельный проект для подразделения.

Такие проекты лучше создавать на конкретный период — например, на год.

Причины:

- **Производительность системы**: проекты с очень большими командами, структурой задач и большим числом списаний рабочего времени могут загружаться медленнее.
- **Актуальность данных**: такой подход позволяет обновлять набор задач с чистого листа — без переименования существующих, закрытия неиспользуемых и т. д.

## Себестоимость проекта

Для проектов, которые включают непроектные работы, следует указывать тип оплаты **Неоплачиваемый**. Они не приносят выручки и относятся к **неутилизируемым**.

При этом у них есть себестоимость: можно оценить затраты компании на непроектные активности и использовать эти данные в аналитике.
