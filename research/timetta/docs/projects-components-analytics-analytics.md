# Вкладка «Аналитика»

Обновлено: 30.08.2025

Вкладка «Аналитика» предназначена для получения быстрого обзора состояния портфеля, программы, организации или отдельного проекта.

## Тренды

Раздел «Тренды» отображает динамику ключевых показателей во времени.

- Доступные масштабы: день, неделя, месяц.
- Выбор валюты: базовая валюта или валюта проекта (доступно только для проектов).

### Показатели

**Часы**

- Факт — количество часов по отметкам времени (в любом состоянии таймшита). Используется ненормализованное значение.
- Оценка — данные из ресурсного плана.
- План — данные из мастер-версии.

**Себестоимость**

- Факт — суммы по проводкам по статье «Себестоимость труда».
- Оценка — данные из ресурсного плана.
- План — данные из мастер-версии.

**Затраты**

- Факт — суммы по проводкам по всем расходным статьям, кроме «Себестоимости труда».
- Оценка — данные из оценки затрат плана.
- План — данные из мастер-версии.

**Выручка**

- Факт — суммы по проводкам по всем доходным статьям.
- Оценка — данные из оценки выручки.
- План — данные из мастер-версии.

### Права доступа

- Для просмотра показателя «Часы» необходимы права на просмотр проекта.
- Для показателей «Себестоимость», «Затраты» и «Выручка» требуются права на просмотр финансовой информации проекта.

## Агрегация данных

Для портфеля, программы и организации данные формируются путем агрегирования информации из связанных проектов.

Если у пользователя нет доступа ко всем проектам, отображаются сведения только по тем проектам, к которым предоставлены права.
