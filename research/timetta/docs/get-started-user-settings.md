# Личные настройки пользователя

## Настройка темы интерфейса

Пользователи могут переключать тему двумя способами:

1. **Быстрое переключение** через иконку в правом верхнем углу:
   - Иконка месяца активирует тёмную тему
   - Иконка солнца возвращает светлую тему

2. **Через вкладку Настройки** в разделе "Тема оформления"

## Мои настройки

Окно "Мои настройки" открывается через меню аватара пользователя в правом верхнем углу экрана.

### Профиль

На этой вкладке пользователи могут загрузить изображение профиля через:
- Кнопку "Выбрать" для выбора файла
- Перетаскивание файла в окно загрузки

### Безопасность

Позволяет изменить пароль (доступно только для локальной аутентификации):
1. Введите текущий пароль
2. Укажите новый пароль и подтвердите его
3. Нажмите "Применить"

### Настройки

Эта вкладка позволяет настроить:

- **Часовой пояс** — отображение времени и даты с учётом выбранного пояса
- **Язык интерфейса** — язык и форматы дат, чисел
- **Способ ввода времени** — три варианта:
  - В часах (по умолчанию)
  - В часах и минутах (чч:мм)
  - В процентах от расписания
- **Тема оформления** — три опции:
  - Светлая
  - Тёмная
  - Системная (подстраивается под ОС)
- **Использовать счётчики времени** — включает функцию счётчиков при работе с таймшитами

### Уведомления

Позволяет выбрать события для email-оповещений:

- Таймшит просрочен
- Задание воркфлоу создано
- Задача назначена
- Дублировать на эл. почту
- Использовать браузерные уведомления

Уведомления отображаются через иконку колокольчика в правом верхнем углу.

### Системные уведомления

Timetta поддерживает push-уведомления операционной системы. Для их работы необходимо разрешить браузеру отправлять уведомления. Доступны инструкции для Chrome, Mozilla, Yandex, Safari и Edge.

### Заместители

На этой вкладке пользователи могут назначить лиц для замещения:

1. Нажмите "Добавить строку"
2. Заполните поля:
   - **Заместитель** — активный пользователь системы (обязательно)
   - **Начало** — дата начала (включительно)
   - **Окончание** — дата окончания (включительно)

Заместители смогут входить от имени пользователя в режиме замещения.

## Навыки

В разделе "Обо мне" вкладки "Профиль" пользователи могут добавить профессиональные навыки:

1. Нажмите "Добавить навыки"
2. Перенесите нужные навыки из левого списка в правый
3. Нажмите "Добавить"
4. Оцените уровень владения по шкале 1-5 звёзд

Администратор системы или менеджер по кадрам также могут заполнять и изменять навыки пользователей.
