# Валюты

## Назначение

**Валюты** представляют справочник денежных единиц, используемых организацией. Система позволяет работать как с одной валютой, так и с несколькими, влияя на отображение символов валюты в заявках на затраты и финансовых данных.

Примечание: Для создания и редактирования записей требуется набор прав администратора с доступом к просмотру и редактированию гранулы "Подразделения".

## Использование

Валютные настройки применяются в следующих компонентах:

- **Проекты** — каждый проект может иметь собственную валюту
- **Заявки на затраты** — используется валюта, назначенная для проекта
- **Финансовые отчеты** — плановые, оценочные, фактические и прогнозные показатели выводятся в валюте проекта и базовой валюте системы

Валюты применяются также в счетах, оплатах и прочих компонентах.

## Создание валюты

Процесс создания валюты:

1. Перейти к компоненте **Валюты**
2. Нажать кнопку **Создать валюту** на панели действий
3. Указать наименование и создать

## Карточка валюты

Карточка содержит основные свойства:

- **Наименование** — название валюты
- **Код** — буквенный код (не может быть изменен после создания; рекомендуется использовать стандарт ISO 4217)
- **Сущность активна** — признак активности (удалить используемую валюту невозможно; необходимо деактивировать)

## Курсы валют

Таблица содержит "Дату" и "Обменный курс", позволяя установить новые значения обменного курса с определенного момента. Для одной даты нельзя указать несколько курсов.

Пример: если базовая валюта — рубль, в курсе доллара указывается количество рублей за 1 USD.
