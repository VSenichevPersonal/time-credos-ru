# Таймшит | Timetta

Обновлено: 28.07.2025

## Назначение

**Таймшит** — это регулярный отчёт о рабочем времени сотрудника за определённый период. С помощью таймшита сотрудники ведут учёт рабочего времени в разрезе клиентов, проектов и задач.

Заполненные таймшиты проходят процесс согласования, а данные из согласованных таймшитов используются для расчёта бизнес-показателей: утилизации, фактической себестоимости проектов и т. д.

## Особенности прав доступа

Для доступа к личным таймшитам в области [Моя работа](https://app.timetta.com/timesheets/current?navigation=my.currentTimesheet) требуется набор прав для роли **Пользователь** с активной гранулой Таймшиты (Просмотр).

Для доступа к таймшитам других пользователей в области [Команда](https://app.timetta.com/timesheets/team-all?navigation=team.timesheetsAll) требуется набор прав для роли **Управление командой** с активной гранулой Таймшиты (как минимум, Просмотр для одной из областей).

Для доступа к таймшитам всех пользователей в системе в области [Финансы](https://app.timetta.com/timesheets/team-all?navigation=finance.timesheetss) требуется набор прав для роли **Управление финансами** (просмотр всех таймшитов включен по умолчанию для указанного набора прав).

## Использование

- **Длительность проекта.** Началом фактической длительности проекта является дата первого списания времени по этому проекту в таймшите.
- **Ресурсный план проекта.** Фактические часы работы по проекту формируются на основании согласованных таймшитов.

## Обзор

![Таймшит](https://timetta.com/entity-assets/9067a124-112f-47f2-8b73-68b401b0d2cf/timesheet-1.png)

1. Панель действий и кнопки **воркфлоу**.
2. Период таймшита и навигация между периодами.
3. Статистика часов за период таймшита.
4. Свойства таймшита (стадия [Жизненного цикла](https://timetta.com/ru/docs/settings/config/life-cycle), срок отправки на согласование, автор).
5. Панель действий: добавление новой строки, удаление, перемещение, копирование, вставка.
6. Строки таймшита.
7. Строки по задачам и работам с таймером.
8. Перечень отметок времени без привязки к задачам.
9. [Заявки на отсутствие](https://timetta.com/ru/docs/time-tracking/components/time-off-request), попадающие в период таймшита.
10. Итоговое время за весь период таймшита включая: заявки на отсутствие, журнал рабочего времени, часы по работам в проекте.

## Журнал рабочего времени

В журнале фиксируется время, запущенное через кнопку таймера в правом верхнем углу.

В форме таймера можно указывать принадлежность:

- К работе на проекте.
- К не связанной с проектом отдельной задаче, в которой пользователь был добавлен в качестве какой-либо роли.

Подробнее — [Задачи](https://timetta.com/ru/docs/projects/conceptions/tasks).
