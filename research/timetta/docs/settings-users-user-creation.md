# Добавление нового пользователя

*Обновлено: 21.08.2025*

> **Примечание**
> Для создания новых пользователей требуется набор прав для роли **Администратор** — как минимум, с активной гранулой «Пользователи: Профили» (права на редактирование).

## Создание пользователя

Чтобы создать пользователя, нужно выполнить следующие шаги:

1. Перейти к компоненте [Пользователи](https://app.timetta.com/settings/users).
2. На панели действий нажать на кнопку **Создать пользователя**. Откроется окно создания:

   ![Диалог создания пользователя](https://timetta.com/entity-assets/a6e2bf1f-f5dd-4443-a075-8258a747ff95/new-user.jpg)

3. Заполнить базовые реквизиты:

   - **Имя** — отображаемое имя пользователя. Должно быть уникальным.
   - **Эл. почта** — почтовый адрес пользователя. Используется для входа в систему. Адрес можно указать только в домене организации. Должен быть уникальным.
   - **Провайдер аутентификации** — единственный доступный пользователю способ входа. Определяет применение политик безопасности. Подробнее — [Настройка политики безопасности](https://timetta.com/ru/docs/settings/system/security-settings).
   - **Набор прав для роли «Пользователь»** — опционально можно присвоить права доступа для роли Пользователь. Выбор из справочника Наборы прав.
   - **Отправить приглашение на вход** — после создания пользователя на адрес эл. почты будет сразу отправлено приглашение на установку пароля и вход в систему.

> **Примечание**
> В Timetta могут входить пользователи, соответствующие следующим критериям:
>
> - Пользователь должен иметь признак **Действующий** в профиле.
> - Должен быть назначен набор прав — как минимум, для роли **Пользователь**.
