# Виды работ

## Назначение

**Виды работ** служат как классификатор при заполнении таймшитов. Они дополняют информацию о рабочем времени, когда это требуется, и указываются в строке таймшита.

Для настройки видов работ необходим набор прав администратора с активной гранулой "Виды работ".

## Создание вида работ

Процесс создания нового вида работ:

1. Перейдите в компоненту **Виды работ**
2. На панели действий нажмите кнопку **Создать вид работ**
3. Введите наименование нового вида работ
4. Нажмите **Создать**

## Настройка вида работ

В карточке вида работ можно настроить следующие параметры:

- **Наименование** — название вида работ
- **Описание** — краткое описание
- **Сущность активна** — признак активности (неактивные записи нельзя выбирать, но они доступны в отчётах)

Если виды работ не используются, колонку можно отключить в настройках шаблона таймшита. Для отдельного пользователя можно ограничить доступные виды работ в карточке пользователя.
