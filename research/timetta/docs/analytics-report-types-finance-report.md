# Тип отчёта «Финансы»

## Назначение

Отчёт «Финансы» объединяет данные из следующих источников:

1. Для оценки и плана — данные проекта и его мастер-версии:
   - Ресурсный план
   - Оценка выручки
   - Оценка затрат

2. Для факта:
   - Проводки

"Отчёт работает по методу начисления. Он использует те же данные, что и Фин. отчёт в карточке проекта." Отчёт содержит базовые исходные данные для анализа, но не заменяет полноценный финансовый отчёт P&L.

## Описание полей

| Поле | Код | Описание |
|------|-----|---------|
| Дата | `Date` | Дата соответствующей сущности — для оценки и плана. Для проводок — дата проводки. |
| Проект | `Project` | Аналитика Проект. |
| Валюта проекта | `ProjectCurrency` | Валюта, указанная в свойствах проекта. |
| Менеджер проекта | `ProjectManager` | Сотрудник, назначенный менеджером проекта. |
| Организация | `Client` | Аналитика Организация. Заказчик проекта. |
| Субподрядчик | `Subcontractor` | Аналитика Организация. Исполнитель работ на субподряде. |
| Менеджер организации | `ClientManager` | Пользователь, назначенный менеджером организации. |
| Программа | `Program` | Аналитика Программа. |
| Менеджер программы | `ProgramManager` | Менеджер программы. |
| Этап | `FirstLevelTask` | Работа проекта верхнего уровня. |
| Учётная статья | `Account` | Финансовая статья или счёт. |
| Описание | `Description` | Комментарии из вкладок проекта: по выручке и затратам, из актов и заявок на затраты. |
| Тип (устарело) | `LineType` | Тип строки отчёта: Выручка, Себестоимость или Затраты. |
| План сумма | `PlannedAmount` | Данные из мастер-плана проекта в валюте проекта. |
| План сумма БВ | `PlannedAmountBC` | Данные из мастер-плана проекта в базовой валюте. |
| Оценка сумма | `EstimatedAmount` | Данные из проекта в валюте проекта. |
| Оценка сумма БВ | `EstimatedAmountBC` | Данные из проекта в базовой валюте. |
| Факт сумма | `ActualAmount` | Данные из проводок в валюте проекта. |
| Факт сумма БВ | `ActualAmountBC` | Данные из проводок в базовой валюте. |
