# Тип отчёта «Ресурсный план»

## Назначение

Этот источник данных объединяет три таблицы: ресурсные планы проектов (оценочные показатели), ресурсные планы мастер-версий проектов (плановые показатели) и согласованные таймшиты (фактические показатели).

### Права доступа

Для доступа к данным отчёта требуется набор прав для роли "Управление проектами" с правами доступа "Просмотр" для соответствующих сущностей.

## Описание полей

| Поле | Код | Описание |
|------|-----|---------|
| Дата | Date | Дата ячейки ресурсного плана или таймшита |
| Ресурс | ResourceName | Наименование ресурса |
| Универсальный ресурс | IsGenericResource | Определяет, является ли ресурс универсальным |
| Ресурсный пул | ResourcePool | Аналитика ресурсного пула |
| Менеджер ресурсного пула | ResourcePoolManager | Аналитика пользователя |
| Пользователь | User | Аналитика пользователя |
| Валюта ставки себестоимости | UserCostRateCurrency | Валюта ставки себестоимости пользователя |
| Руководитель | Supervisor | Аналитика пользователя, руководитель ресурса |
| Организация | OrganizationId | Аналитика организации |
| Менеджер организации | OrganizationManager | Аналитика пользователя |
| Программа | Program | Аналитика программы |
| Менеджер программы | ProgramManager | Пользователь, назначенный менеджером программы |
| Проект | Project | Аналитика проекта |
| Валюта проекта | ProjectCurrency | Валюта финансовых показателей проекта |
| Менеджер проекта | ProjectManager | Пользователь, назначенный менеджером проекта |
| Работа | ProjectTask | Работа проекта |
| Этап | FirstLevelTask | Работа верхнего уровня |
| Роль | Role | Аналитика роли |
| План Часы | PlannedHours | Часы, запланированные в ресурсном плане мастер-версии |
| План Себес. | PlannedCost | Сумма часов по ставкам себестоимости |
| План Себес. БВ | PlannedCostBC | Себестоимость по матрице, приведённой к базовой валюте |
| План Себес. ВП | PlannedCostPC | Себестоимость по матрице, приведённой к валюте проекта |
| Оценка Часы | EstimatedHours | Часы из ресурсного плана |
| Оценка Себес. | EstimatedCost | Себестоимость из ресурсного плана |
| Оценка Себес. БВ | EstimatedCostBC | Оценка себестоимости в базовой валюте |
| Оценка Себес. ВП | EstimatedCostPC | Оценка себестоимости в валюте проекта |
| Факт Часы | ActualHours | Часы по проектам из согласованных ТШ + часы отсутствий |
| Факт Часы Норм. | ActualHoursNormalized | Нормализованные часы из согласованных таймшитов |
| Факт Себес. | ActualCost | Ненормализованная себестоимость согласованных таймшитов |
| Факт Себес. БВ | ActualCostBC | Ненормализованная себестоимость в базовой валюте |
| Факт Себес. ВП | ActualCostPC | Ненормализованная себестоимость в валюте проекта |
| Факт Выручка T&M | ActualRevenueTM | Факт выручки по тарифам в валюте тарифа |
| Часы Расписание | HoursSchedule | Сумма часов по расписанию за выбранный период |
| Часы FTE | HoursFteSchedule | Сумма часов по расписанию со свойством "По умолчанию" |
| Часы Всего за пер. | HoursTotalOverPeriod | Часы, списанные пользователем на проекты и отсутствия |
