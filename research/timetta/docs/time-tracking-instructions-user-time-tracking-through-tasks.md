# Учёт времени через задачи

## Создать отметку времени

Для создания отметки времени, выполните следующие шаги:

1. Перейдите в раздел "Все задачи"
2. Откройте карточку нужной задачи
3. На вкладке "Учёт времени" нажмите кнопку "Добавить отметку"
4. В открывшемся окне выберите задачу или работу для привязки времени
5. При необходимости укажите дополнительные параметры (продолжительность, вид работ, комментарий)
6. Нажмите кнопку "Создать"

**Примечание:** Доступность задач и работ зависит от прав пользователя. Задача или работа доступны для выбора, если текущий пользователь — исполнитель, наблюдатель или создатель, а также если у него есть права на списание времени.

## Управление отметкой времени

После создания отметки она отобразится на вкладке "Учёт времени" карточки задачи.

Для редактирования отметки нажмите на иконку карандаша (появляется при наведении курсора). Время можно задать двумя способами:

- Запустить таймер с помощью зелёной кнопки "Запустить счётчик" (доступно только для текущего дня)
- Ввести вручную

## Расположение созданной отметки

Созданную отметку времени можно найти:

- В карточке привязанной задачи на вкладке "Учёт времени"
- В разделе "Журнал рабочего времени" таймшита, который соответствует дате созданной отметки
