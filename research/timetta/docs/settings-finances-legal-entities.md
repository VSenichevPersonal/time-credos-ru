# Юридические лица

## Назначение

Юридические лица представляют собой справочник организаций, предназначенный для ведения списка компаний, сотрудники которых являются пользователями текущего абонента Timetta. Это может включать материнские компании, субсидиарные или партнёрские организации.

## Использование

Справочник применяется в нескольких компонентах системы:

- **Юридические лица**: хранение информации об организациях
- **Пользователи**: каждому пользователю назначается юридическое лицо из справочника
- **Проекты**: проектам присваивается юридическое лицо
- **Центры затрат**: каждый центр затрат связан с одним юридическим лицом

Использование юридических лиц рекомендуется при управлении несколькими компаниями в рамках одного абонента, наличии внутренних взаиморасчётов между организациями или привлечении сотрудников из разных компаний на один проект.

## Добавление юридических лиц

При создании абонента система автоматически генерирует одно юридическое лицо по умолчанию. Пользователи с соответствующими правами могут создавать новые записи и редактировать существующие.

## Карточка юридического лица

Карточка содержит следующие реквизиты:

- **Юридическое лицо используется по умолчанию** — применяется автоматически к новым пользователям
- **Наименование** — название организации
- **Код** — идентификатор юридического лица
- **Менеджер** — ответственный за организацию
- **Валюта** — основная валюта организации
- **Описание** — дополнительная информация
- **Сущность активна** — статус доступности для выбора
- **Дополнительные поля** — настраиваемые атрибуты
