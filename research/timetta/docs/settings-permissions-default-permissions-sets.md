# Стандартные наборы прав

## Назначение

В большинстве случаев стандартных наборов прав достаточно для разграничения доступа пользователей. Эти наборы создаются по умолчанию при развёртывании абонента в Timetta.

## Стандартные наборы прав

| Роль | Набор прав | Описание |
|------|-----------|---------|
| Пользователь | Пользователь | Даёт права на вход в систему, доступ к "Моя работа", "Аналитика" и "Поддержка". Позволяет создавать и редактировать таймшиты, заявки на отсутствия и затраты. |
| Управление проектами | Менеджер проектов | Даёт доступ к области "Проекты" с возможностью создания и редактирования проектов, где пользователь назначен менеджером. |
| Управление проектами | Администратор проектов | Даёт доступ к области "Проекты" с полными правами на все проекты. |
| Управление командой | Руководитель | Даёт доступ к области "Команда" с просмотром таймшитов и заявок подчинённых пользователей. |
| Управление командой | Менеджер по кадрам | Даёт доступ к области "Команда" с просмотром таймшитов и заявок всех пользователей. |
| Управление клиентами | Менеджер клиентов | Даёт доступ к области "Клиенты" для управления контактами, взаимодействиями, сделками и кампаниями. |
| Управление финансами | Финансовый менеджер | Даёт доступ к области "Финансы" для просмотра финансовых сведений по всей компании. |
| Управление биллингом | Биллинг-менеджер | Даёт доступ к области "Биллинг" для управления счетами. |
| Управление ресурсами | Ресурсный менеджер | Даёт доступ к области "Ресурсы" для редактирования планов бронирования и исполнения запросов. |
| Администратор | Системный администратор | Даёт доступ к области "Настройки" для управления настройками Timetta. |

## Роли пользователей

### Пользователь

Базовый сотрудник, выполняющий операции заполнения таймшитов, заявок на отсутствие и затраты. Имеет доступ к областям "Моя работа", "Аналитика" и "Поддержка".

### Управление командой

Роль для линейных руководителей и модераторов системы. Включает контроль заполнения таймшитов, мониторинг загрузки сотрудников, модерацию системы и оценку навыков команды.

### Управление проектами

Присваивается менеджерам проектов и сотрудникам проектного офиса. Отвечает за создание, редактирование проектов, формирование команд, планов, ресурсных планов и контроль показателей.

### Управление ресурсами

Для ресурсных менеджеров и линейных руководителей. Охватывает распределение ресурсов между проектами и контроль их загрузки.

### Управление клиентами

Предназначена для менеджеров клиентов. Включает фиксирование взаимодействий, отслеживание сделок и управление активностями отдела продаж.

### Управление финансами

Для финансовых менеджеров и контролёров. Предусматривает контроль финансовых сведений, закрытие учётных периодов, согласование планов и управление проводками.

### Управление биллингом

Для финансовых менеджеров. Охватывает выставление счетов и контроль оплаты по T&M-проектам.

### Администратор

Выполняет задачи администрирования: создание пользователей, настройка прав доступа, управление маршрутами согласования, расписаниями и бизнес-правилами.

**Примечание:** Если стандартных наборов недостаточно, администратор может создать дополнительные наборы прав.
