# Создание запроса на подбор и актуализацию ресурса

## Создание запроса на подбор

**Запросы на подбор** доступны только для универсальных ресурсов. Новое бронирование добавляется к существующему бронированию ресурса.

Процесс создания:

1. Добавить универсальный ресурс в команду проекта (вкладка **Команда**)
2. Опционально: назначить ресурс на работы проекта (вкладка **Структура работ**)
3. Опционально: внести плановые часы в ресурсный план (вкладка **Ресурсы**)
4. На вкладке **Бронирование** нажать кнопку **+** в колонке **Запрос** для универсального ресурса
5. Заполнить параметры запроса и нажать **Создать**
6. В открывшейся карточке нажать **Отправить запрос** — запрос направляется ресурсному менеджеру

**Примечание:** Запрос можно создать и без привязки к проекту через раздел **Ресурсы** → подраздел **Все запросы**.

## Создание запроса на актуализацию

Актуализация — запрос по конкретному пользователю для корректировки планов загрузки уже добавленных ресурсов. Результат заменяет существующее бронирование.

Процесс:

1. Перейти на вкладку **Бронирование** в карточке проекта
2. Нажать **+** в колонке **Запрос** для забронированного ресурса
3. Заполнить поля и создать запрос
4. Отправить запрос ресурсному менеджеру

## Массовое создание запросов

Для создания нескольких запросов одновременно:

1. На вкладке **Бронирование** отметить верхний чекбокс для всех ресурсов или выбрать конкретные строки
2. Нажать кнопку **Запросить (число выбранных ресурсов)**

По выбранным ресурсам будут сформированы запросы в состоянии «Черновик» согласно текущему ресурсному плану.
