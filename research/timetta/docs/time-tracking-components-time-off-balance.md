# Расчёт баланса отсутствий

## Назначение

Компонент "Баланс отсутствий" регистрирует начисления и использование дней или часов по различным типам отсутствий сотрудников. Доступен в разделе "Команда" и требует набора прав "Управление командой" с активной гранулой "Начисления и списания отсутствий".

## Использование

Баланс отсутствий применяется в двух основных контекстах:

1. **Заявки на отсутствие** — расчёт и отображение баланса для каждой заявки
2. **Профиль сотрудника** — просмотр баланса за различные периоды времени

## Обзор компоненты

Раздел представляет таблицу с записями об изменениях в балансе. Каждая запись содержит:

- **Тип**: "Начисление" (увеличивает доступные дни/часы) или "Использование" (уменьшает)
- **Пользователь**: сотрудник, для которого создана запись
- **Тип отсутствия**: категория отсутствия
- **Единица учёта**: дни или часы (определяется типом)
- **Способ создания**: автоматический или ручной (не редактируется)
- **Дата**: дата создания записи
- **Значение**: количество начисленных или использованных дней/часов

## Способы создания записей

Записи с типом "Использование" создаются автоматически на основе согласованных заявок на отсутствие. При переводе заявки в статус "Черновик" соответствующая запись удаляется автоматически.

Записи с типом "Начисление" создаются автоматически при установленной политике начисления для типа отсутствий.

Администраторы с необходимыми правами могут создавать записи вручную.
