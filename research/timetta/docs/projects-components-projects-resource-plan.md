# Ресурсный план

Обновлено: 22.07.2025

## Назначение

**Ресурсный план** проекта предназначен для оценки часов по проекту в разрезе членов команды, работ и календарных периодов.

Для проектов с [автоматическим типом планирования](https://timetta.com/ru/docs/projects/conceptions/planning/auto-mode) ресурсный план может быть заполнен автоматически при заполнении структуры работ.

> **Права доступа**
>
> Для доступа к ресурсному плану требуется набор прав для роли `**Управление проектами**` с активной гранулой Ресурсный план.

## Обзор

Ресурсный план находится на вкладке **Ресурсы** проекта:

![Ресурсный план проекта](https://timetta.com/entity-assets/fa6ca5db-5db0-4ef5-8b7f-1f64278afad0/rp.jpg)

- Переключатель режима представления. Доступно два представления — **Прогноз** и **Оценка**.
- Навигация по периодам — перемещение вперёд и назад по календарю.
- Быстрый переход на конкретную дату.
- Выбор единицы планирования (в часах, процентах от расписания, FTE, днях).
- Выбор масштаба планирования (по дням, неделям, месяцам).
- Очистка ресурсного плана.
- Отмена и повторное выполнение последнего действия.
- Переключение представления сроков работ.
- Настройка представления — можно управлять списком колонок и отображаемыми значениями в таблице.
- Наименование ресурса (пользователя или универсального).
- Проектная роль ресурса.
- Список работ (назначений) члена команды.
- Общая сумма плановых часов по строке (за все периоды, не только за отображаемый).
- Плановые трудозатраты в выбранной единице планирования.
- Общая сумма часов по всем членам команды проекта за все периоды.
- Сумма часов по всем ресурсам за выбранную единицу планирования (день, неделю, месяц).
