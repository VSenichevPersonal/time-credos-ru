# Использование технологий искусственного интеллекта в Timetta

## AI-ассистент: интеллектуальная поддержка и консультирование

**AI-ассистент Timetta** — встроенный инструмент для решения вопросов пользователей. Он выполняет функции технической поддержки и бизнес-консультанта.

### Ключевые задачи AI-ассистента:

1. **Составление пошаговых инструкций**
   Пользователь может задать вопрос в свободной форме. Ассистент генерирует чёткое руководство по действиям за несколько секунд.

2. **Оперативные ответы на вопросы**
   Ассистент объясняет, как работают конкретные функции Timetta.

3. **Решение проблем**
   Пользователь описывает возникшую задачу или ошибку. Ассистент предлагает решения на основе знаний о системе.

4. **Анализ данных и прогнозирование** (скоро)
   На основе данных пользователя ассистент предоставит аналитическую информацию:
   - Количество запланированных часов по проекту
   - Список доступных сотрудников из ресурсного пула
   - Оценка степени готовности проекта

5. **Поддержка интеграций** (скоро)
   Ассистент помогает формировать API-запросы:
   - Находит системные названия сущностей и полей
   - Автоматически генерирует запрос с нужными параметрами

### Технологическая основа

AI-ассистент построен на комбинации современных технологий:

- **RAG-архитектура (Retrieval-Augmented Generation)**
  Ответы формируются на основе актуальных данных из базы знаний и документации Timetta.

- **Языковые модели**
  Используются "современные отечественные и Open Source модели, включая Llama, Qwen и YandexGPT 5".

- **Векторная база данных (Vector DB)**
  Позволяет находить семантические связи в документации.

## Генерация контента в CRM

В модуле **Timetta Clients** доступна функция интеллектуальной генерации текста для упрощения коммуникации с клиентами.

### Принцип работы

1. Вместо статического шаблона пользователь создаёт текстовый запрос для большой языковой модели (LLM).
2. Система автоматически генерирует тему и текст письма.
3. Для персонализации в промптах можно использовать шаблоны **Scriban** для подстановки контекстных данных.

### Назначение

Функция экономит время на создании типовых писем, сохраняя релевантность и персонализацию каждого сообщения.
