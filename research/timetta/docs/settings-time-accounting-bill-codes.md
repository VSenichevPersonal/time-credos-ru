# Коды оплаты

**Коды оплат** — это классификатор рабочего времени, используемый для расчёта почасовой оплаты или премий.

## Назначение

Система позволяет создавать коды (например: «Норма», «Переработка», «Работа в выходные») для классификации отработанного времени. По итогам месяца можно построить отчёт для расчёта зарплаты и/или премий. В настройках кода можно указать поправку себестоимости в процентах.

## Создание кода оплаты

Требуется набор прав для роли `Администратор` с активной гранулой «Коды оплаты».

Процесс создания:
1. Перейти к компоненте **Коды оплаты**
2. На панели действий нажать **Создать код оплаты**
3. Указать наименование и код нового кода оплат
4. Нажать **Создать**

## Настройка кода оплаты

Форма содержит следующие поля:

- **Наименование** — название кода оплаты (обязательное)
- **Код** — системный код для кода оплаты
- **Поправка себестоимости** — процент, на который изменяется ставка себестоимости
- **Описание** — краткое описание
- **Активно** — признак активности

**Расчёт:** При себестоимости 1000 р. и поправке 10%, итоговая себестоимость = 1000 + (1000 × 0,1) = 1100 р.

## Отображение

Чтобы добавить поле в карточку таймшита, активируйте "Показывать Код оплаты" в настройках шаблона таймшита.
