# Модель ресурсов

## Типы ресурсов

Timetta поддерживает три основных типа ресурсов:

1. **Сотрудник (Employee)** — конкретный человек в организации
2. **Универсальный ресурс (Generic)** — условный ресурс для предварительного планирования
3. **Вакансия (Vacancy)** — открытая должность, заполняемая впоследствии

## Свойства ресурсов

Ресурсы характеризуются следующими параметрами:

- **Роль** — основная проектная функция (например, разработчик, аналитик)
- **Специализация** — предметная область в рамках роли
- **Тип занятости** — формат взаимодействия (штатный, подрядчик, совместитель)
- **Юридическое лицо** — для учёта и финансовой аналитики
- **Локация** — местоположение для расчёта региональных ставок
- **Ресурсный пул** — функциональное объединение (отдел, практика)
- **Уровень** — общая квалификация (Junior, Middle, Senior, Lead)
- **Грейд** — детализация уровня для карьерной модели
- **Руководитель** — непосредственный начальник
- **Подразделение** — структурное подразделение

## Ключевые компоненты

**Роль** представляет базовую функцию и связана с требованиями к компетенциям.

**Специализация** уточняет роль и отражает профессиональную экспертизу, может включать дополнительные требования к компетенциям.

**Уровень** определяет общую категорию квалификации профессиональной зрелости специалиста.

**Грейд** детализирует уровень квалификации и используется для управления ставками и карьерным развитием. Это опциональный параметр.
