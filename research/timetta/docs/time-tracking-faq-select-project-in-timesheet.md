# Какие проекты доступны в таймшите

Обновлено: 24.09.2025

Для списания времени на работу проекта необходимо выбрать её в таймшите.

Работа доступна для выбора при соблюдении следующих условий:

1. Проект должен быть действующим, то есть не находиться в статусах **Отменен** или **Архивирован**.

2. Автор таймшита должен входить в команду проекта (напрямую как пользователь или в составе подразделения). При этом член команды (пользователь или подразделение) не должен быть отключен.

3. Автор таймшита должен быть назначен хотя бы на одну работу в проекте (явно или через назначение **Всей команды**).

> **Примечание.** Самая частая причина того, что проект не отображается в таймшите, — у пользователя нет назначения на работы проекта.

4. Хотя бы одна назначенная работа должна быть с активной опцией **«Разрешено списание времени»**. Если учёт времени ведётся на главную работу проекта (без создания отдельных задач), в свойствах проекта должна быть включена опция **«Разрешить списание времени без выбора работ»**.

5. Работа должна быть активной, т. е. не отмечена как **Выполненная**.

> **Примечание.** Настройки жизненного цикла могут быть изменены администратором.
