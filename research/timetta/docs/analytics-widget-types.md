# Типы виджетов

Обновлено: 31.10.2023

## Назначение

Виджет — это информационный блок на панели мониторинга.

Типы виджетов:

- Диаграммы: гистограммы, график, круговая диаграмма.
- Индикаторы: спидометр и значение.
- Таблица: аналогичен отчету системы.
- Фрейм: позволяет отобразить внешнюю страницу (например, документ Google Spreadsheet) или собственный HTML код.

## Гистограмма

На оси X располагаются категории, колонки соответствуют значениям. Если выбрано несколько значений или легенда (детализация), то в каждой категории отображается несколько колонок.

Позволяет выбрать:

- Одно поле категорий. На примере для категорий выбран «Пользователь».
- Одно значение + легенду или несколько значений. На примере для легенды выбрано «Состояние», в качестве значения используются «Часы».

## Гистограмма с накоплением

На оси X располагаются категории, колонки соответствуют значениям. Если выбрано несколько значений или легенда (детализация), то каждая колонка разбивается на сегменты.

Настройка аналогична гистограмме.

![Гистограмма с накоплением](https://timetta.com/entity-assets/9122fe3e-8a5d-4f3b-91af-eca1dcf4134f/accumulated-histogram.png)

## Нормированная гистограмма

На оси X располагаются категории, колонки всегда равны 100%. Если выбрано несколько значений или легенда (детализация), то каждая колонка разбивается на сегменты в %% от доли значения.

Настройка аналогична гистограмме.

![Нормированная гистограмма](https://timetta.com/entity-assets/9122fe3e-8a5d-4f3b-91af-eca1dcf4134f/normalized-histogram.png)

## График

На оси X располагаются категории (чаще шкала дат). Точки линии соответствуют значениям. Если выбрано несколько значений или легенда (детализация), то отображается несколько линий.

Настройка аналогична гистограмме.

![График](https://timetta.com/entity-assets/9122fe3e-8a5d-4f3b-91af-eca1dcf4134f/graph.png)

## Круговая диаграмма

Легенда соответствует сегментам, а значение — размеру сегмента.

Позволяет выбрать:

- Одно поле легенды. На примере выбрано поле «Вид работ».
- Одно поле значения. На примере выбрано поле «Всего часов».

![Круговая диаграмма](https://timetta.com/entity-assets/9122fe3e-8a5d-4f3b-91af-eca1dcf4134f/pie-chart.png)
