# Методы финансового учёта

_Обновлено: 25.08.2025_

Есть два метода финансового учёта: кассовый и метод начисления.

- По кассовому методу признание выручки происходит в момент поступления денег на счёт
- По методу начисления выручка признаётся в момент завершения работ (например, при подписании акта).

## Характеристики методов

**Кассовый метод (Cash Accounting)** — доход признаётся в день поступления денег (или иного имущества, работ, услуг). Расходами признаются затраты по факту их оплаты.

**Метод начисления (Accrual Accounting)** — доходы и расходы фиксируются в момент, когда фактически произошла операция, независимо от того, когда были получены или выплачены деньги.

## Сравнение методов

Важно учитывать не только выручку, но и затраты — например, себестоимость труда.

Для расчёта показателей нельзя смешивать методы. Например, нельзя считать прибыль, используя выручку по кассовому методу, а затраты — по методу начисления.

Кассовый метод проще в применении. Метод начисления точнее отражает реальную экономическую ситуацию на проекте.

Примеры:

- Необходимо посчитать рентабельность при закрытии проекта, но клиент платит с отсрочкой. По кассовому методу придётся ждать поступления оплаты, по методу начисления расчёт можно выполнить сразу.
- Длинный проект без выделенных финансовых этапов: по кассовому методу невозможно подвести итоги года.
- Оплата взаимозачётом: деньги не поступают, и кассовый метод неприменим.

Кассовый метод подходит для небольших компаний и простых моделей. Метод начисления более универсален.

## Реализация в Timetta

### Учёт по методу начисления

Выручка по проекту:

- Оценка/план/прогноз — через календарь и список в карточке проекта. Даты оценки — оценочные даты признания выручки.
- Факт — через акты.

Затраты:

- Оценка/план/прогноз — через календарь и список в карточке проекта. Даты оценки — оценочные даты признания затрат.
- Факт — через заявки на затраты. Дата заявки — дата признания затрат, обычно совпадает с датой понесения.

Себестоимость:

- Оценка/план/прогноз — через ресурсный план проекта
- Факт — через таймшиты

Показатели привязаны к датам начисления — реального задействования трудового ресурса.

Финансовый отчет (P&L) отражает экономику по методу начисления.

### Учёт по кассовому методу

Выставление счетов и получение оплаты:

- Оценка/план/прогноз — через календарь и список счетов и оплаты в карточке проекта
- Факт — через счета в области «Биллинг»
