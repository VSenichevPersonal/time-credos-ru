# Учёт переработок

## Назначение

**Учёт переработок** является инструментом для контроля и расчёта стоимости часов, превышающих плановые объёмы. Система автоматически учитывает часы, отработанные в следующих ситуациях:

- В праздничные дни
- В выходные дни
- После окончания рабочего дня
- В другие нестандартные периоды

## Два основных способа учёта переработок

### 1. Нормализация себестоимости

Этот метод применяется, когда зарплата сотрудника фиксированная и переработки не подлежат оплате. Подробная информация доступна в разделе "Нормализация себестоимости".

### 2. Коды оплат

Этот метод используется, когда переработки требуют оплаты. С помощью кодов оплат можно:
- Управлять типами переработок
- Задавать коэффициенты к себестоимости
- Учитывать их при заполнении таймшитов

Коды оплат также служат дополнительной аналитикой при построении отчётов для определения превышения бюджета по зарплате.

## Доступность пользователям

### Использование нормализации себестоимости

Установите правило нормализации в карточке пользователя на вкладке **Себестоимость** для игнорирования переработок. Подробнее см. раздел "Управление пользователями".

### Использование кодов оплат

Добавьте столбец **Коды оплаты** в шаблон таймшита. По умолчанию в таймшите доступен полный список кодов оплат, заданный в настройках системы.
