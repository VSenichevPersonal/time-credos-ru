# AI Bot Architecture (исследовательский интерес)

**Дата:** 2026-06-21
**Объект:** ai-bot.timetta.com

---

## Трёхуровневая модель знаний

### Tier 1 — Контекст диалога
- Все предыдущие сообщения в текущем чате
- Поддерживает логическую связь, уточнения

### Tier 2 — Статические знания (встроенные)
- Предварительно обучен на структурированных знаниях
- Включает: логику таймшитов, жизненные циклы, RateMatrix, P&L
- Основан на: официальной документации, FAQ, типовых сценариях
- "Встроены в конфигурацию", "статический набор правил"
- Обновляется командой платформы

### Tier 3 — Внешние источники (RAG, при активации)
- Официальная документация (timetta.com/ru/docs)
- Внутренняя Wiki (если подключена)
- Справка и FAQ
- **Индексация: раз в 24 часа**
- При ответе ссылается на конкретные страницы

## Ключевой вывод

Бот даёт детальные ответы потому что платформа вложилась в:
1. Глубокое эмбеддирование документации в конфигурацию модели
2. Ежедневную авто-индексацию help-центра
3. Статические шаблоны знаний по ключевым доменам

Это НЕ живой доступ к данным. Это НЕ RAG реального времени.
Это гибрид: статические эмбеддинги + суточный индекс документации.

---

## Сценарии работы (2 режима)

### Режим 1: Без внешних источников
- Только статические знания + история диалога
- Если данных недостаточно — честно говорит
- Не выдумывает. Всегда доступен.

### Режим 2: С поиском по документации
- + поиск по timetta.com/ru/docs
- + внутренняя Wiki (если подключена)
- Авто-выбор режима системой классификации

## Вспомогательные модели

| Модель | Тип | Задача |
|--------|-----|--------|
| Классификатор запросов | Малая LLM / ML | Определяет тип вопроса, выбирает сценарий |
| Re-ranker | Семантический поиск | Оценка релевантности найденных фрагментов |
| Embedding model | text-embedding-ada-002 / bge-small | Векторизация текста для поиска |
| Фильтр безопасности | Бинарный классификатор + правила | Проверка на безопасность и политики |

## Обучение

- ❌ НЕ fine-tuned на документации
- ✅ Чистый RAG: статические эмбеддинги + суточный индекс
- Базовые знания встроены в конфигурацию (не в веса модели)
- Безопасность: бинарный классификатор + правила, не только промпт
